## Was dies bereitstellt

Der Traffic-Generator ist eine speziell entwickelte Azure-VM, die realistischen Angriffsverkehr, Aufklärungsscans, Bot-Simulation und API-Missbrauchsmuster gegen einen F5 Distributed Cloud HTTP Load Balancer erzeugt. Er validiert, dass WAF-Richtlinien, Bot Defense, API Security und Client-Side Defense korrekt konfiguriert sind, indem er genau den Datenverkehr generiert, den diese Funktionen erkennen und blockieren sollen.

Alle Tools werden während der Terraform-Bereitstellung über cloud-init vorinstalliert. Der Datenverkehr ist in Suiten organisiert, die einzeln oder nacheinander mit dem enthaltenen `runner.sh`-Orchestrator ausgeführt werden können.
