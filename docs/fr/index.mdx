## Ce que cela fournit

Le générateur de trafic est une VM Azure spécialement conçue qui produit du trafic d'attaque réaliste, des scans de reconnaissance, de la simulation de bots et des schémas d'abus d'API contre un répartiteur de charge HTTP F5 Distributed Cloud. Il valide que les politiques WAF, la défense contre les bots, la sécurité API et la défense côté client sont correctement configurées en générant exactement le trafic que ces fonctionnalités sont conçues pour détecter et bloquer.

Tous les outils sont préinstallés via cloud-init lors du provisionnement Terraform. Le trafic est organisé en suites qui peuvent être exécutées individuellement ou en séquence à l'aide de l'orchestrateur `runner.sh` inclus.

## Catégories de suites de trafic

| Suite | Description | Fonctionnalité F5 XC validée |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10, mode API SQLMap, découverte de paramètres, fuzzing d'endpoints | Sécurité API |
| [bot-simulation](/traffic-generator/05-suites/#bot-simulation) | Chrome headless, Puppeteer stealth, automatisation Playwright, crawling rapide | Défense contre les bots |
| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | Comportement du cache, effet de troupeau, pool de connexions, multiplexage HTTP/2 | Intégration CDN |
| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA, bruteforce OTP, manipulation JWT, SSRF, injection NoSQL, IDOR | Sécurité API |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | Skimmer de carte, formjacker, keylogger, cryptominer, détournement DOM | Défense côté client |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | DoS par requêtes groupées, récursion profonde, injection SQL, abus d'introspection | Sécurité API (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | Force brute, injection de commandes, CSRF, inclusion de fichiers, SQLi, XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | Manipulation DOM, injection de scripts inline, charges utiles de skimming de type Magecart | Défense côté client |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | Contournement de connexion SQLi, XSS, IDOR, accès administrateur, accès fichier par octet nul | WAF, Défense contre les bots |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | Tactiques ATT&CK : reconnaissance, accès initial, accès aux identifiants, exfiltration | WAF, Défense contre les bots, Sécurité API |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | ZAP, Nikto, Nuclei, scan de vulnérabilités Nmap, rapport OWASP combiné | WAF, Scan d'applications web |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | Montée en concurrence, charge soutenue, test de pics, découverte de points de rupture | DDoS, Limitation de débit |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap, Masscan, Gobuster, Subfinder, force brute de répertoires | WAF / Défense contre les bots |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA, BOPLA, BFLA, contournement de limitation de débit, secret JWT faible | Sécurité API |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | SSLScan, sslyze, analyse de configuration TLS testssl.sh | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | Trafic HTTP légitime à haut volume pour les tests de référence et de charge | Tous |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | Encodage URL/HTML/Unicode multi-couches, encodage imbriqué mixte, TE chunked, injection d'en-têtes | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | Injection SQL, XSS, injection de commandes, traversée de chemin, Nikto, Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi, XSS, traversée de chemin contre les endpoints de test WAF de F5 DemoApp | WAF |
