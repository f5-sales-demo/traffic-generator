## 必要條件

### Azure 訂閱

一個有效的 Azure 訂閱，具備 Contributor 角色或同等權限以建立以下資源：

- 資源群組
- 虛擬網路和子網路
- 網路安全性群組
- 公用 IP 位址
- 虛擬機器（預設為 Standard_F16s_v2）

### Azure CLI

安裝並驗證 Azure CLI 以取得您的訂閱 ID：

```bash
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

az login

az account show --query '{name:name, id:id}' -o table
```

複製 `id` 值——這是 Terraform 所需的 `subscription_id`。

### Terraform

需要 Terraform >= 1.5：

```bash
terraform version
```

如果尚未安裝，請使用 HashiCorp APT 儲存庫：

```bash
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
```

### SSH 金鑰對

用於 VM 存取的 SSH 金鑰對。建議使用 Ed25519：

```bash
ssh-keygen -t ed25519 -f ~/.ssh/traffic-generator-key -N ""
```

### 網路存取

Traffic Generator VM 在佈建過程中需要對外 HTTPS 存取以從以下來源下載工具：

- GitHub Releases（nuclei、dalfox、ffuf、gobuster、feroxbuster、subfinder、httpx、amass）
- PyPI（sqlmap、scapy、impacket、arjun、pwntools、mitmproxy、sslyze、playwright）
- npm registry（puppeteer、puppeteer-extra-plugin-stealth）
- Ubuntu APT 儲存庫（nikto、nmap、masscan、hydra、tshark 及其他系統套件）
- Git 儲存庫（testssl.sh、recon-ng、spiderfoot、SecLists）

如果您的 Azure 訂閱有強制執行對外防火牆規則或 NSG 限制，請確保在初始佈建期間（15-25 分鐘）允許 HTTPS（連接埠 443）連線至這些來源。

### 預先部署的元件

在部署 Traffic Generator 之前，您應該已具備：

1. 已部署且可連線的 **Origin Server**（請參閱 [origin-server 文件](https://f5xc-salesdemos.github.io/origin-server/03-deploy/)）
2. 已設定 origin server 作為後端的 **F5 XC HTTP Load Balancer**
3. 已附加至負載平衡器的 **WAF 原則**（用於 WAF 流量驗證）

Terraform 變數 `target_fqdn` 應設定為 F5 XC 負載平衡器的公用網域。

:::tip
實驗室未使用時，請使用 `terraform destroy` 以避免持續產生費用。相關程序請參閱[清除資源](../08-teardown/)。
:::

:::note
Standard_F16s_v2（32 GiB RAM）為預設值，因為在套件執行期間多個工具會同時運行。瀏覽器自動化（Playwright、Puppeteer）搭配無頭 Chromium 是主要的記憶體消耗來源。較小的 F 系列 SKU 可用於非瀏覽器套件，但在執行機器人模擬時可能會較為吃緊。
:::
