## 本工具提供的功能

流量產生器是一個專門建構的 Azure 虛擬機，能夠針對 F5 Distributed Cloud HTTP 負載平衡器產生真實的攻擊流量、偵察掃描、Bot 模擬和 API 濫用模式。它透過產生這些功能所設計偵測和阻擋的確切流量，來驗證 WAF 政策、Bot Defense、API Security 和 Client-Side Defense 是否正確配置。

所有工具均在 Terraform 佈建期間透過 cloud-init 預先安裝。流量被組織為套件，可以使用內建的 `runner.sh` 協調器個別或依序執行。

## 流量套件分類

| 套件 | 描述 | 驗證的 F5 XC 功能 |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10、SQLMap API 模式、參數探索、端點模糊測試 | API Security |
| [bot-simulation](/traffic-generator/05-suites/#bot-simulation) | 無頭 Chrome、Puppeteer 隱匿模式、Playwright 自動化、快速爬取 | Bot Defense |
| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | 快取行為、雷鳴群效應、連線池、HTTP/2 多工 | CDN 整合 |
| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA、OTP 暴力破解、JWT 操縱、SSRF、NoSQL 注入、IDOR | API Security |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | 信用卡側錄器、表單劫持器、鍵盤記錄器、加密挖礦程式、DOM 劫持 | Client-Side Defense |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | 批次查詢 DoS、深層遞迴、SQL 注入、內省濫用 | API Security (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | 暴力破解、命令注入、CSRF、檔案包含、SQLi、XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | DOM 操縱、內聯腳本注入、Magecart 風格的側錄酬載 | Client-Side Defense |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | SQLi 登入繞過、XSS、IDOR、管理員存取、空位元組檔案存取 | WAF、Bot Defense |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | ATT&CK 戰術：偵察、初始存取、憑證存取、資料外洩 | WAF、Bot Defense、API Security |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | ZAP、Nikto、Nuclei、Nmap 弱點掃描、綜合 OWASP 報告 | WAF、Web App 掃描 |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | 並行量遞增、持續負載、尖峰測試、斷點探索 | DDoS、速率限制 |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap、Masscan、Gobuster、Subfinder、目錄暴力破解 | WAF / Bot Defense |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA、BOPLA、BFLA、速率限制繞過、JWT 弱密鑰 | API Security |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | SSLScan、sslyze、testssl.sh TLS 配置分析 | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | 高流量合法 HTTP 流量，用於基線和負載測試 | 全部 |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | 多層 URL/HTML/Unicode 編碼、混合巢狀編碼、分塊 TE、標頭注入 | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | SQL 注入、XSS、命令注入、路徑遍歷、Nikto、Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi、XSS、路徑遍歷，針對 F5 DemoApp WAF 測試端點 | WAF |
