جميع الأدوات المدرجة أدناه مثبتة مسبقاً أثناء عملية التهيئة عبر cloud-init. لا حاجة لأي تثبيت يدوي بعد النشر.

## اختبار تجاوز جدار حماية تطبيقات الويب (WAF)

| الأداة | طريقة التثبيت | أمر مثال | تُستخدم بواسطة المجموعة |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

GoTestWAF (من Wallarm) هو إطار عمل معياري لتقييم جدار حماية تطبيقات الويب يضاعف الحمولات عبر مُرمِّزات متعددة (URL، Base64، JSUnicode) ومناطق وضع متنوعة (الجسم، الترويسة، معامل URL، الكوكي). أما waf-bypass (من Nemesida) فيرسل الحمولات بترميز Base64 وHTML-ENTITY وUTF-16 عبر مناطق ARGS وBODY وCOOKIE وHEADER.

## المستوى الكامل فقط

يتم تثبيت هذه الأدوات فقط عند تعيين `tool_tier = "full"`:

| الأداة | طريقة التثبيت | أمر مثال | تُستخدم بواسطة المجموعة |
|---|---|---|---|
| ZAP (OWASP Zed Attack Proxy) | ghlatest | `zap-cli quick-scan -s all -r https://TARGET` | web-app-attacks |
| Metasploit Framework | installer script | `msfconsole -q -x "use auxiliary/scanner/http/http_version; set RHOSTS TARGET; run; exit"` | reconnaissance |

:::note
يضيف ZAP وMetasploit ما يقارب 1.5 جيجابايت إلى حجم القرص ومن 5 إلى 10 دقائق إلى وقت التهيئة. استخدم المستوى الكامل فقط إذا كنت تحتاج هذه الأدوات تحديداً لعرض توضيحي.
:::

## مرجع طرق التثبيت

| الطريقة | الوصف |
|---|---|
| apt | مثبتة من مستودعات APT لنظام Ubuntu 24.04 عبر `apt-get install` |
| ghlatest | أحدث ملف ثنائي يتم تنزيله من إصدارات GitHub باستخدام `ghlatest.sh` و `install-release.sh` |
| uv | مثبتة كأداة Python باستخدام مدير الحزم `uv` |
| pip | مثبتة كحزمة Python باستخدام `pip install` في Python النظام |
| npm | مثبتة كحزمة Node.js عامة باستخدام `npm install -g` |
| go install | مُجمَّعة من المصدر باستخدام `go install` ومثبتة في `/usr/local/bin` |
| git clone | المستودع مُستنسَخ في مجلد `/opt/` |
| git clone + pip | المستودع مُستنسَخ مع تثبيت اعتماديات Python |
| installer script | سكريبت تثبيت مقدم من المورد يتم تنفيذه أثناء cloud-init |
