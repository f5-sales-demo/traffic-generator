جميع الأدوات المدرجة أدناه مثبتة مسبقاً أثناء عملية التهيئة عبر cloud-init. لا حاجة لأي تثبيت يدوي بعد النشر.

## اختبار تطبيقات الويب

| الأداة | طريقة التثبيت | أمر مثال | المجموعة المستخدمة |
|---|---|---|---|
| nikto | apt | `nikto -h https://TARGET -maxtime 120s` | web-app-attacks |
| sqlmap | apt | `sqlmap --batch -u "https://TARGET/dvwa/vulnerabilities/sqli/?id=1"` | web-app-attacks, api-attacks |
| nuclei | ghlatest | `nuclei -u https://TARGET -severity medium,high,critical` | web-app-attacks |
| dalfox | ghlatest | `dalfox url "https://TARGET/search?q=test" --silence` | web-app-attacks |
| ffuf | ghlatest | `ffuf -u https://TARGET/FUZZ -w wordlist.txt -mc all -fc 404` | api-attacks |
| gobuster | ghlatest | `gobuster dir -u https://TARGET -w /opt/SecLists/Discovery/Web-Content/common.txt` | reconnaissance |

## اختبار تجاوز جدار حماية تطبيقات الويب

| الأداة | طريقة التثبيت | أمر مثال | المجموعة المستخدمة |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

يُعد GoTestWAF (من Wallarm) إطار عمل معياري لتقييم جدران حماية تطبيقات الويب، حيث يضاعف الحمولات عبر مُرمِّزات متعددة (URL، Base64، JSUnicode) ومناطق وضع مختلفة (الجسم، الترويسة، معامل URL، ملف تعريف الارتباط). أما waf-bypass (من Nemesida) فيرسل حمولات بترميز Base64 وHTML-ENTITY وUTF-16 عبر مناطق ARGS وBODY وCOOKIE وHEADER.

## المستوى الكامل فقط

يتم تثبيت هذه الأدوات فقط عندما يكون `tool_tier = "full"`:

| الأداة | طريقة التثبيت | أمر مثال | المجموعة المستخدمة |
|---|---|---|---|
| ZAP (OWASP Zed Attack Proxy) | ghlatest | `zap-cli quick-scan -s all -r https://TARGET` | web-app-attacks |
| Metasploit Framework | installer script | `msfconsole -q -x "use auxiliary/scanner/http/http_version; set RHOSTS TARGET; run; exit"` | reconnaissance |

:::note
يضيف ZAP وMetasploit حوالي 1.5 جيجابايت إلى المساحة المستخدمة على القرص ومن 5 إلى 10 دقائق إلى وقت التهيئة. استخدم المستوى الكامل فقط إذا كنت تحتاج هذه الأدوات تحديداً لعرض توضيحي.
:::

## مرجع طرق التثبيت

| الطريقة | الوصف |
|---|---|
| apt | مثبت من مستودعات APT لنظام Ubuntu 24.04 عبر `apt-get install` |
| ghlatest | أحدث ملف ثنائي محمّل من إصدارات GitHub باستخدام `ghlatest.sh` و`install-release.sh` |
| uv | مثبت كأداة Python باستخدام مدير الحزم `uv` |
| pip | مثبت كحزمة Python باستخدام `pip install` في Python الخاص بالنظام |
| npm | مثبت كحزمة Node.js عامة باستخدام `npm install -g` |
| go install | مُجمَّع من المصدر باستخدام `go install` ومثبت في `/usr/local/bin` |
| git clone | مستودع منسوخ إلى مجلد `/opt/` |
| git clone + pip | مستودع منسوخ مع تثبيت اعتماديات Python |
| installer script | سكريبت تثبيت مقدم من المورّد يتم تنفيذه أثناء cloud-init |
