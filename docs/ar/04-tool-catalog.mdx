جميع الأدوات المدرجة أدناه مثبتة مسبقاً أثناء تهيئة cloud-init. لا حاجة لأي تثبيت يدوي بعد النشر.

## اختبار تجاوز جدار حماية تطبيقات الويب

| الأداة | طريقة التثبيت | أمر مثالي | المجموعة المستخدمة |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

GoTestWAF (من Wallarm) هو إطار عمل معياري لتقييم جدران حماية تطبيقات الويب يضاعف الحمولات عبر عدة مشفرات (URL، Base64، JSUnicode) ومناطق وضع متعددة (body، header، URL param، cookie). waf-bypass (من Nemesida) يرسل الحمولات بترميز Base64 وHTML-ENTITY وUTF-16 عبر مناطق ARGS وBODY وCOOKIE وHEADER.

## الفئة الكاملة فقط

يتم تثبيت هذه الأدوات فقط عند تعيين `tool_tier = "full"`:

| الأداة | طريقة التثبيت | أمر مثالي | المجموعة المستخدمة |
|---|---|---|---|
| ZAP (OWASP Zed Attack Proxy) | ghlatest | `zap-cli quick-scan -s all -r https://TARGET` | web-app-attacks |
| Metasploit Framework | installer script | `msfconsole -q -x "use auxiliary/scanner/http/http_version; set RHOSTS TARGET; run; exit"` | reconnaissance |

:::note
يضيف ZAP وMetasploit ما يقارب 1.5 جيجابايت إلى حجم القرص و5-10 دقائق إلى وقت التهيئة. استخدم الفئة الكاملة فقط إذا كنت بحاجة تحديداً إلى هذه الأدوات للعرض التوضيحي.
:::

## مرجع طرق التثبيت

| الطريقة | الوصف |
|---|---|
| apt | مثبت من مستودعات APT لنظام Ubuntu 24.04 عبر `apt-get install` |
| ghlatest | أحدث ملف ثنائي يتم تنزيله من إصدارات GitHub باستخدام `ghlatest.sh` و `install-release.sh` |
| uv | مثبت كأداة Python باستخدام مدير الحزم `uv` |
| pip | مثبت كحزمة Python باستخدام `pip install` في Python النظام |
| npm | مثبت كحزمة Node.js عامة باستخدام `npm install -g` |
| go install | مترجم من المصدر باستخدام `go install` ومثبت في `/usr/local/bin` |
| git clone | المستودع مستنسخ إلى مجلد `/opt/` |
| git clone + pip | المستودع مستنسخ وتبعيات Python مثبتة |
| installer script | سكربت تثبيت مقدم من المورد يتم تنفيذه أثناء cloud-init |
