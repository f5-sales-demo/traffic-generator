## ما الذي يوفره هذا

مولّد حركة المرور هو جهاز افتراضي Azure مُصمم خصيصًا لإنتاج حركة مرور هجومية واقعية، ومسح استطلاعي، ومحاكاة بوتات، وأنماط إساءة استخدام واجهات API ضد موازن أحمال HTTP في F5 Distributed Cloud. يتحقق من أن سياسات WAF وBot Defense وAPI Security وClient-Side Defense مُهيأة بشكل صحيح من خلال توليد حركة المرور الدقيقة التي صُممت هذه الميزات لاكتشافها وحظرها.

يتم تثبيت جميع الأدوات مسبقًا عبر cloud-init أثناء التزويد بواسطة Terraform. يتم تنظيم حركة المرور في مجموعات يمكن تشغيلها بشكل فردي أو بالتتابع باستخدام المُنسق المُضمن `runner.sh`.

## فئات مجموعات حركة المرور

| المجموعة | الوصف | ميزة F5 XC التي يتم التحقق منها |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10، وضع SQLMap API، اكتشاف المعلمات، تشويش نقاط النهاية | API Security |
| [bot-simulation](/traffic-generator/05-suites/#bot-simulation) | Chrome بدون واجهة رسومية، Puppeteer stealth، أتمتة Playwright، زحف سريع | Bot Defense |
| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | سلوك التخزين المؤقت، القطيع المتدافع، مجمع الاتصالات، تعدد إرسال HTTP/2 | CDN Integration |
| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA، هجوم القوة الغاشمة على OTP، التلاعب بـ JWT، SSRF، حقن NoSQL، IDOR | API Security |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | سارق بطاقات، اختطاف نماذج، مسجل مفاتيح، تعدين خفي، اختطاف DOM | Client-Side Defense |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | هجوم حرمان الخدمة بالاستعلامات المجمعة، التكرار العميق، حقن SQL، إساءة استخدام الاستبطان | API Security (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | القوة الغاشمة، حقن الأوامر، CSRF، تضمين الملفات، SQLi، XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | التلاعب بـ DOM، حقن نصوص برمجية مضمنة، حمولات سرقة بيانات بأسلوب Magecart | Client-Side Defense |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | تجاوز تسجيل الدخول بـ SQLi، XSS، IDOR، الوصول للمسؤول، الوصول للملفات بالبايت الفارغ | WAF, Bot Defense |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | تكتيكات ATT&CK: استطلاع، وصول أولي، وصول لبيانات الاعتماد، تسريب البيانات | WAF, Bot Defense, API Security |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | ZAP، Nikto، Nuclei، فحص ثغرات Nmap، تقرير OWASP مجمع | WAF, Web App Scanning |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | تصعيد التزامن، حمل مستدام، اختبار الذروة، اكتشاف نقطة الانهيار | DDoS, Rate Limiting |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap، Masscan، Gobuster، Subfinder، هجوم القوة الغاشمة على المجلدات | WAF / Bot Defense |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA، BOPLA، BFLA، تجاوز تحديد المعدل، مفتاح JWT ضعيف | API Security |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | SSLScan، sslyze، تحليل تهيئة TLS بواسطة testssl.sh | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | حركة مرور HTTP شرعية عالية الحجم لاختبار خط الأساس والحمل | الكل |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | ترميز متعدد الطبقات URL/HTML/Unicode، ترميز متداخل مختلط، TE مجزأ، حقن الترويسات | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | حقن SQL، XSS، حقن الأوامر، اجتياز المسارات، Nikto، Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi، XSS، اجتياز المسارات ضد نقاط نهاية اختبار WAF في F5 DemoApp | WAF |
