## ما الذي يوفره هذا المشروع

مولّد حركة المرور هو جهاز افتراضي Azure مُصمّم خصيصاً لإنتاج حركة مرور هجومية واقعية وعمليات مسح استطلاعية ومحاكاة بوتات وأنماط إساءة استخدام واجهات API ضد موزّع حمل HTTP في F5 Distributed Cloud. يتحقق من أن سياسات WAF وحماية البوتات وأمان واجهات API وحماية جانب العميل مُعدّة بشكل صحيح من خلال توليد حركة المرور الدقيقة التي صُمّمت هذه الميزات لاكتشافها وحظرها.

يتم تثبيت جميع الأدوات مسبقاً عبر cloud-init أثناء التوفير بواسطة Terraform. يتم تنظيم حركة المرور في مجموعات يمكن تشغيلها بشكل فردي أو بالتتابع باستخدام المنسّق المُضمّن `runner.sh`.

## فئات مجموعات حركة المرور

| المجموعة | الوصف | ميزة F5 XC التي يتم التحقق منها |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | قائمة OWASP API العشر الأوائل، وضع SQLMap لواجهات API، اكتشاف المعاملات، تشويش نقاط النهاية | أمان واجهات API |
| [bot-simulation](/traffic-generator/05-suites/#bot-simulation) | Chrome بدون واجهة، تخفي Puppeteer، أتمتة Playwright، زحف سريع | حماية البوتات |
| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | سلوك التخزين المؤقت، تأثير القطيع المفاجئ، مجمع الاتصالات، تعدد إرسال HTTP/2 | تكامل CDN |
| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA، هجوم القوة الغاشمة على OTP، التلاعب بـ JWT، SSRF، حقن NoSQL، IDOR | أمان واجهات API |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | سارق البطاقات، اختطاف النماذج، مسجّل المفاتيح، تعدين العملات المشفرة، اختطاف DOM | حماية جانب العميل |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | حرمان الخدمة بالاستعلامات المجمّعة، التكرار العميق، حقن SQL، إساءة استخدام الاستبطان | أمان واجهات API (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | القوة الغاشمة، حقن الأوامر، CSRF، تضمين الملفات، SQLi، XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | التلاعب بـ DOM، حقن النصوص البرمجية المضمّنة، حمولات سرقة بأسلوب Magecart | حماية جانب العميل |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | تجاوز تسجيل الدخول بـ SQLi، XSS، IDOR، الوصول للمسؤول، الوصول للملفات بالبايت الفارغ | WAF، حماية البوتات |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | تكتيكات ATT&CK: الاستطلاع، الوصول الأولي، الوصول لبيانات الاعتماد، تسريب البيانات | WAF، حماية البوتات، أمان واجهات API |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | فحص الثغرات بـ ZAP وNikto وNuclei وNmap، تقرير OWASP مُجمّع | WAF، فحص تطبيقات الويب |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | تدرج التزامن، الحمل المستمر، اختبار الذروة، اكتشاف نقطة الانهيار | حماية DDoS، تحديد المعدل |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap، Masscan، Gobuster، Subfinder، هجوم القوة الغاشمة على المجلدات | WAF / حماية البوتات |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA، BOPLA، BFLA، تجاوز تحديد المعدل، مفتاح JWT ضعيف | أمان واجهات API |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | تحليل تكوين TLS بـ SSLScan وsslyze وtestssl.sh | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | حركة مرور HTTP مشروعة بحجم كبير لاختبار خط الأساس والحمل | الكل |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | ترميز متعدد الطبقات URL/HTML/Unicode، ترميز متداخل مختلط، نقل مُجزّأ TE، حقن الترويسات | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | حقن SQL، XSS، حقن الأوامر، اجتياز المسار، Nikto، Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi، XSS، اجتياز المسار ضد نقاط اختبار WAF في F5 DemoApp | WAF |
