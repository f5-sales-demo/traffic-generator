## ما الذي يوفره هذا

مولّد حركة المرور هو جهاز افتراضي Azure مصمم خصيصاً لإنتاج حركة مرور هجومية واقعية وعمليات استطلاع ومحاكاة للبوتات وأنماط إساءة استخدام واجهات برمجة التطبيقات ضد موازن أحمال HTTP في F5 Distributed Cloud. يتحقق من أن سياسات WAF وBot Defense وAPI Security وClient-Side Defense مُعدّة بشكل صحيح من خلال توليد حركة المرور الدقيقة التي صُممت هذه الميزات لاكتشافها وحظرها.

يتم تثبيت جميع الأدوات مسبقاً عبر cloud-init أثناء توفير Terraform. يتم تنظيم حركة المرور في مجموعات يمكن تشغيلها بشكل فردي أو بالتتابع باستخدام المنسّق المضمّن `runner.sh`.

## فئات مجموعات حركة المرور

| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10، وضع SQLMap لواجهات API، اكتشاف المعلمات، تشويش نقاط النهاية | API Security |

| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA، القوة الغاشمة لـ OTP، التلاعب بـ JWT، SSRF، حقن NoSQL، IDOR | API Security |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | سارق بطاقات، اختطاف نماذج، راصد لوحة مفاتيح، تعدين عملات مشفرة، اختطاف DOM | Client-Side Defense |

| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | القوة الغاشمة، حقن الأوامر، CSRF، تضمين الملفات، SQLi، XSS | WAF |

| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | فحص الثغرات بـ ZAP وNikto وNuclei وNmap، تقرير OWASP مجمع | WAF, Web App Scanning |
