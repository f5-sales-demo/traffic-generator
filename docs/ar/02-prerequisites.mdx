## المتطلبات

### اشتراك Azure

اشتراك Azure نشط مع دور المساهم (Contributor) أو أذونات مكافئة لإنشاء:

- مجموعات الموارد
- الشبكات الافتراضية والشبكات الفرعية
- مجموعات أمان الشبكة
- عناوين IP العامة
- الأجهزة الافتراضية (Standard_F16s_v2 افتراضيًا)

### واجهة سطر أوامر Azure

قم بتثبيت واجهة سطر أوامر Azure والمصادقة عليها لاسترداد معرّف الاشتراك الخاص بك:

```bash
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

az login

az account show --query '{name:name, id:id}' -o table
```

انسخ قيمة `id` -- هذا هو `subscription_id` المطلوب من قبل Terraform.

### Terraform

يتطلب Terraform بإصدار 1.5 أو أعلى:

```bash
terraform version
```

إذا لم يكن مثبتًا، استخدم مستودع HashiCorp APT:

```bash
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
```

### زوج مفاتيح SSH

زوج مفاتيح SSH للوصول إلى الجهاز الافتراضي. يُوصى باستخدام Ed25519:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/traffic-generator-key -N ""
```

### الوصول إلى الشبكة

يتطلب جهاز مولّد حركة المرور الافتراضي وصولاً صادرًا عبر HTTPS أثناء التزويد لتنزيل الأدوات من:

- إصدارات GitHub ‏(nuclei، dalfox، ffuf، gobuster، feroxbuster، subfinder، httpx، amass)
- PyPI ‏(sqlmap، scapy، impacket، arjun، pwntools، mitmproxy، sslyze، playwright)
- سجل npm ‏(puppeteer، puppeteer-extra-plugin-stealth)
- مستودعات Ubuntu APT ‏(nikto، nmap، masscan، hydra، tshark، وحزم النظام الأخرى)
- مستودعات Git ‏(testssl.sh، recon-ng، spiderfoot، SecLists)

إذا كان اشتراك Azure الخاص بك يفرض قواعد جدار حماية صادرة أو قيود NSG، فتأكد من السماح بـ HTTPS (المنفذ 443) إلى هذه المصادر أثناء نافذة التزويد الأولية (15-25 دقيقة).

### المكونات المنشورة مسبقًا

قبل نشر مولّد حركة المرور، يجب أن يكون لديك:

1. **خادم الأصل** منشورًا وقابلاً للوصول (راجع [وثائق خادم الأصل](https://f5xc-salesdemos.github.io/origin-server/03-deploy/))
2. **موازن حمل HTTP من F5 XC** مُهيّأ مع خادم الأصل كخلفية
3. **سياسة WAF** مرفقة بموازن الحمل (للتحقق من صحة حركة مرور WAF)

يجب تعيين متغير Terraform المسمى `target_fqdn` إلى النطاق العام لموازن حمل F5 XC.

:::tip
استخدم `terraform destroy` عندما لا يكون المختبر قيد الاستخدام لتجنب الرسوم المستمرة. راجع [الإزالة](../08-teardown/) للاطلاع على الإجراء.
:::

:::note
Standard_F16s_v2 (32 جيجابايت ذاكرة وصول عشوائي) هو الإعداد الافتراضي لأن العديد من الأدوات تعمل بشكل متزامن أثناء تنفيذ مجموعة الاختبارات. أتمتة المتصفح (Playwright، Puppeteer) مع Chromium بدون واجهة رسومية هي المستهلك الرئيسي للذاكرة. ستعمل وحدات SKU الأصغر من سلسلة F مع مجموعات الاختبارات التي لا تستخدم المتصفح، ولكنها قد تكون محدودة عند تشغيل محاكاة الروبوتات.
:::
