كل مجموعة زيارات هي دليل ضمن `/opt/traffic-generator/suites/` يحتوي على برامج نصية مرقمة يتم تنفيذها بالترتيب بواسطة `runner.sh`. يقبل كل برنامج نصي اسم النطاق المؤهل الكامل (FQDN) للهدف كوسيط أول ويكتب المخرجات إلى كل من stdout ودليل النتائج.

## api-attacks

**البرامج النصية:** 4 | **المدة المقدرة:** 10-17 دقيقة

اختبار OWASP API Security Top 10 ضد VAmPI عبر F5 XC. يسجل مستخدم اختبار، ويحصل على رمز مصادقة، ثم يختبر BOLA، والمصادقة بالقوة الغاشمة، والتعرض المفرط للبيانات، وتخويل مستوى الوظائف المعطل، وأنماط SSRF، وحقن SQL ضد نقاط نهاية REST API، واكتشاف المعاملات المخفية، وتشويش نقاط النهاية. يستهدف تطبيق VAmPI.

**ميزة F5 XC:** أمان API / اكتشاف API

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-vampi-owasp-top10.sh` | curl, jq | مجموعة اختبار شاملة لـ OWASP API Top 10. يسجل مستخدم اختبار، ويحصل على رمز مصادقة، ثم يختبر BOLA (الوصول إلى بيانات مستخدمين آخرين)، والمصادقة بالقوة الغاشمة، والتعرض المفرط للبيانات، وتخويل مستوى الوظائف المعطل، وأنماط SSRF، وفحص أخطاء تكوين الأمان. |
| 02 | `02-sqlmap-api.sh` | sqlmap | حقن SQL ضد نقاط نهاية REST API لـ VAmPI: البحث عن المستخدم، وتسجيل الدخول (POST)، والتسجيل (POST) مع حمولات JSON. |
| 03 | `03-arjun-param-discovery.sh` | arjun | اكتشاف المعاملات المخفية عبر نقاط نهاية Juice Shop وDVWA وVAmPI. يكتشف معاملات الاستعلام غير الموثقة التي قد تقبل حمولات الحقن. |
| 04 | `04-ffuf-api-fuzz.sh` | ffuf | تشويش ثنائي المراحل: اكتشاف نقاط النهاية باستخدام قائمة كلمات لمسارات API الشائعة (api, swagger, graphql, admin, .env, .git) عبر جميع بادئات التطبيقات، ثم تشويش أساليب HTTP (GET, POST, PUT, DELETE, PATCH, OPTIONS, TRACE, PROPFIND) ضد نقاط النهاية الرئيسية. |

**سلوك F5 XC المتوقع:** يجب أن يقوم اكتشاف API بتعيين نقاط نهاية VAmPI API وتحديد أنماط الطلبات غير المعتادة. يجب أن تُبلِّغ سياسات أمان API عن محاولات الوصول غير المصرح بها، وحمولات الحقن في أجسام JSON، وأنماط التعداد. سيُظهر مخزون API في F5 XC نقاط النهاية المكتشفة مع أعداد الطلبات.

## bot-simulation

**البرامج النصية:** 4 | **المدة المقدرة:** 5-10 دقائق

نشاط المتصفح الآلي وأنماط الزحف الشبيهة بالروبوتات عبر F5 XC. يتضمن Chrome بدون واجهة، وPuppeteer مع إضافة التخفي، والزحف السريع، والتفاعل الآلي مع النماذج. يستهدف جميع التطبيقات.

**ميزة F5 XC:** الدفاع ضد الروبوتات

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-headless-chrome.sh` | playwright | تصفح Chromium بدون واجهة: يحمّل الهدف، ويتنقل بين مسارات التطبيقات، ويستخرج عناوين الصفحات والمحتوى، ويلتقط لقطات شاشة. يولّد زيارات قائمة على المتصفح بدون إضافات تخفي (قابلة للكشف كأتمتة). |
| 02 | `02-puppeteer-stealth.sh` | puppeteer, puppeteer-extra-plugin-stealth | تصفح خفي باستخدام puppeteer-extra مع إضافة التخفي. يحاول التهرب من كشف الروبوتات عن طريق تعديل خصائص navigator وWebGL وبصمات المتصفح الأخرى. |
| 03 | `03-rapid-crawl.sh` | curl, wget | طلبات HTTP عالية التردد باستخدام curl وwget بتتابع سريع. يحاكي سلوك الكاشط بدون تأخير بين الطلبات مع تدوير عنيف لـ User-Agent. |
| 04 | `04-form-interaction.sh` | playwright | ملء النماذج وإرسالها آليًا ضد تسجيل دخول DVWA، وتسجيل Juice Shop، ونقاط نهاية VAmPI API باستخدام واجهات التفاعل مع النماذج في Playwright. |

**سلوك F5 XC المتوقع:** يجب أن يصنف الدفاع ضد الروبوتات الزيارات على أنها آلية بناءً على تحليل بصمة المتصفح، وتوقيت الطلبات، ونتائج تحدي JavaScript. يجب اكتشاف Chrome بدون واجهة وبدون تخفي على الفور. تختبر اختبارات Puppeteer الخفي ما إذا كانت تقنيات التهرب المتقدمة تتجاوز الكشف. ستُظهر لوحة معلومات الدفاع ضد الروبوتات فئات تصنيف الروبوتات وإجراءات التخفيف.

## cdn-load-testing

**البرامج النصية:** 18 | **المدة المقدرة:** 20-30 دقيقة

اختبار سلوك ذاكرة التخزين المؤقت لشبكة CDN وحماية الأصل. يتضمن الإنتاجية الأساسية، وعزل سلسلة الاستعلام، وتنوع accept-encoding، ومحاكاة القطيع المندفع، وتجاوز POST/PUT، وتصادم مفتاح الذاكرة المؤقتة، وGET المشروط، وطلب النطاق، ومحاكاة التطهير، واستنفاد تجمع الاتصالات، وتحسين keepalive، وعبء مصافحة TLS، وتخزين الكائنات الكبيرة مؤقتًا، وتجاوز الفشل متعدد الأصول، وتحديد المعدل، واختبار نسبة gzip، وتعدد إرسال HTTP/2، ومعيار kraken المجمع. يستهدف تطبيق محاكي CDN.

**ميزة F5 XC:** تكامل CDN

## crapi-exploits

**البرامج النصية:** 16 | **المدة المقدرة:** 15-25 دقيقة

استغلال تحديات OWASP crAPI (واجهة برمجة تطبيقات سخيفة تمامًا). يتضمن التسجيل/المصادقة، وBOLA لموقع المركبة، وBOLA لتقارير الميكانيكي، والقوة الغاشمة لـ OTP، وتعرض بيانات الميكانيكيين، والتعرض المفرط للبيانات، والتخصيص الجماعي للطلبات، والتحقق من صحة قسيمة SSRF، وحقن NoSQL، والتلاعب بـ JWT، ولوحة معلومات IDOR، وتجاوز المصادقة الثنائية، والوصول إلى API الداخلي، وتلوث معاملات جانب الخادم، وحذف المستخدم، ورفع فيديو IDOR. يستهدف تطبيق crAPI.

**ميزة F5 XC:** أمان API

## csd-demo-attacks

**البرامج النصية:** 5 (JavaScript) | **المدة المقدرة:** 3-5 دقائق

هجمات حقن JavaScript من جانب العميل. تتضمن حقن كاشط البطاقات، ومستولي النماذج، ومسجل لوحة المفاتيح، ومُعدِّن العملات المشفرة، واختطاف DOM. تختبر هجمات سلسلة التوريد من جانب العميل على طراز Magecart التي يكتشفها ويحظرها الدفاع من جانب العميل في F5 XC. يستهدف تطبيق CSD التجريبي.

**ميزة F5 XC:** الدفاع من جانب العميل

## demoapp-attacks

**البرامج النصية:** 2 | **المدة المقدرة:** 2-3 دقائق

حمولات هجوم مستهدفة ضد نقاط نهاية اختبار WAF المدمجة في F5 DemoApp (`/WAF/SQL`، `/WAF/XSS`، `/WAF/DIR`). ترسل حمولات حقن SQL وXSS واجتياز المسار مباشرة إلى نقاط النهاية المصممة لإظهار سلوك حظر WAF. استخدم هذه المجموعة عند استهداف خادم محتوى DemoApp بدلاً من تطبيقات الخادم الأصلي القياسية.

**ميزة F5 XC:** WAF (جدار حماية تطبيقات الويب)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-sqli-waf-endpoint.sh` | curl, python3 | 15 حمولة حقن SQL ضد `/WAF/SQL?age=` تشمل UNION SELECT، وDROP TABLE، وWAITFOR DELAY، وxp_cmdshell، وتجاوز قائم على التعليقات، وحمولات المنطق البولياني. |
| 02 | `02-xss-waf-endpoint.sh` | curl, python3 | 18 حمولة XSS ضد `/WAF/XSS?update=` تشمل وسوم script، ومعالجات الأحداث (onerror, onload, ontoggle, onfocus)، وeval بتشفير base64، وتعبير style، والحمولات متعددة الأشكال، واستخراج ملفات تعريف الارتباط. |

**سلوك F5 XC المتوقع:** يجب أن يحظر WAF الطلبات التي تحتوي على حمولات حقن SQL وXSS ويعيد صفحة حظر. ستُظهر لوحة معلومات أحداث الأمان فئات الانتهاكات ومطابقات التوقيعات لكل طلب محظور.

## dvga-exploits

**البرامج النصية:** 9 | **المدة المقدرة:** 8-12 دقيقة

استغلال ثغرات خاصة بـ GraphQL ضد تطبيق Damn Vulnerable GraphQL Application. يتضمن حرمان الخدمة بالاستعلامات المجمعة، والتكرار العميق، وتكرار الحقول، وحقن SQL عبر الفلاتر، وXSS عبر createPaste، وحرمان الخدمة القائم على الأسماء المستعارة، وإساءة استخدام الاستبطان، وتجاوز التخويل، والكشف عن المعلومات. يستهدف تطبيق DVGA.

**ميزة F5 XC:** أمان API (GraphQL)

## dvwa-exploits

**البرامج النصية:** 14 | **المدة المقدرة:** 15-25 دقيقة

اختبار OWASP Top 10 ضد تطبيق Damn Vulnerable Web Application. يتضمن القوة الغاشمة، وحقن الأوامر، وتغيير كلمة مرور CSRF، وتضمين الملفات، ورفع الملفات، وحقن SQL (الأعمى)، وحقن SQL (union)، وXSS (DOM)، وXSS (المنعكس)، وXSS (المخزن)، وتجاوز CAPTCHA، ومعرفات الجلسات الضعيفة، وCORS غير الآمن، وإعادة التوجيه المفتوحة. يستهدف تطبيق DVWA.

**ميزة F5 XC:** WAF

## javascript-exploits

**البرامج النصية:** 3 | **المدة المقدرة:** 3-5 دقائق

أنماط حقن البرامج النصية من جانب العميل والتلاعب بـ DOM التي تُطلق كشف الدفاع من جانب العميل. تتضمن حقن البرامج النصية المضمنة التي تحاكي كاشطات على طراز Magecart، والتلاعب بـ DOM عبر Chromium بدون واجهة، ومحاكاة البرامج النصية لطرف ثالث. يستهدف تطبيق CSD التجريبي.

**ميزة F5 XC:** الدفاع من جانب العميل (CSD)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-inline-script-injection.sh` | curl | يرسل طلبات تحتوي على حمولات JavaScript مضمنة مصممة لمحاكاة كاشطات بطاقات الائتمان على طراز Magecart وبرامج استخراج البيانات النصية. |
| 02 | `02-dom-manipulation.sh` | playwright | يستخدم Chromium بدون واجهة لتحميل صفحة CSD التجريبية وحقن تعديلات DOM: إضافة وسوم script، وتعديل إجراءات النماذج، وإدراج إطارات iframe غير مرئية تستخرج بيانات النماذج. |
| 03 | `03-third-party-script-sim.sh` | curl, playwright | يحاكي برامج نصية لطرف ثالث مخترقة عن طريق حقن طلبات تشير إلى مصادر JavaScript خارجية ومحاولة تحميل برامج نصية غير مصرح بها في سياق الصفحة. |

**سلوك F5 XC المتوقع:** يجب أن يكتشف الدفاع من جانب العميل تعديلات البرامج النصية غير المصرح بها على الصفحة المحمية. ستُظهر لوحة معلومات CSD تنبيهات لمصادر البرامج النصية الجديدة، وعناصر DOM المعدلة، ومحاولات استخراج البيانات. يتحقق هذا من وظائف المرحلة الثانية لـ CSD.

<Aside type="note">
مجموعة javascript-exploits مصممة للعمل جنبًا إلى جنب مع تطبيق CSD التجريبي على الخادم الأصلي. تأكد من تمكين مسار `/csd-demo/` على الخادم الأصلي قبل تشغيل هذه المجموعة.
</Aside>

## juice-shop-exploits

**البرامج النصية:** 12 | **المدة المقدرة:** 10-18 دقيقة

استغلال تحديات OWASP Juice Shop. يتضمن تجاوز تسجيل الدخول بحقن SQL، وحقن SQL في البحث بـ union، وXSS DOM المنعكس، وXSS المخزن عبر API، والوصول إلى سلة IDOR، والوصول إلى قسم المسؤول، والتعليقات المزورة، والوصول إلى الملفات بالبايت الفارغ، وXSS المنعكس في البحث، وحقن رأس HTTP، والتسجيل المتكرر، والقوة الغاشمة لتسجيل الدخول. يستهدف تطبيق Juice Shop.

**ميزة F5 XC:** WAF، الدفاع ضد الروبوتات

## mitre-attack

**البرامج النصية:** 8 | **المدة المقدرة:** 10-15 دقيقة

محاكاة تكتيكات إطار عمل MITRE ATT&CK. يتضمن الاستطلاع (المسح الخارجي)، والوصول الأولي (حشو بيانات الاعتماد)، والتنفيذ (حقن الأوامر)، والوصول إلى بيانات الاعتماد (كلمات المرور الافتراضية)، والاكتشاف (تعداد الأدلة)، ومحاكاة الحركة الجانبية، والجمع (كشط البيانات)، ومحاكاة الاستخراج. يعيّن كل برنامج نصي إلى تكتيك MITRE ATT&CK محدد لتقارير تغطية التهديدات المنظمة. يستهدف جميع التطبيقات.

**ميزة F5 XC:** WAF، الدفاع ضد الروبوتات، أمان API

**ميزة F5 XC:** WAF، فحص تطبيقات الويب

مسح الشبكة وتعداد الخدمات والقوة الغاشمة للأدلة ضد البنية التحتية المستهدفة. يتضمن كشف خدمات nmap، ومسح منافذ masscan، والقوة الغاشمة للأدلة، وتعداد النطاقات الفرعية، وتحديد بصمة تقنيات الويب، واستطلاع DNS. يستهدف جميع التطبيقات.

**سلوك F5 XC المتوقع:** يجب أن يكتشف WAF ويسجل نشاط المسح. قد يصنف الدفاع ضد الروبوتات الطلبات المتتابعة السريعة على أنها آلية. ستُظهر لوحة معلومات أحداث الأمان توقيعات الاستطلاع وسلاسل User-Agent لأدوات المسح.

| 03 | `03-testssl.sh` | testssl.sh | تقييم TLS كامل باستخدام إطار عمل testssl.sh. يختبر دعم البروتوكول وتفضيلات التشفير وتحليل الرؤوس والثغرات المعروفة. |

**سلوك F5 XC المتوقع:** يولّد مسح SSL العديد من مصافحات TLS مع اقتراحات مجموعات تشفير غير معتادة. في حين أن F5 XC لا يحظرها عادةً، يكون نمط الزيارات مرئيًا في سجلات الوصول. القيمة الأساسية هي التحقق من أن تكوين TLS لـ F5 XC يلبي أفضل ممارسات الأمان.

## traffic-generation

**البرامج النصية:** 4 | **المدة المقدرة:** 5-10 دقائق (قابلة للتكوين)

زيارات HTTP شرعية عالية الحجم للقياس الأساسي واختبار الحمل. تُنشئ أنماط زيارات طبيعية في تحليلات F5 XC للمقارنة مع زيارات مجموعة الهجوم. يستهدف جميع التطبيقات.

**ميزة F5 XC:** الكل (مقارنة أساسية)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-baseline-http.sh` | curl | طلبات HTTP GET متتابعة لجميع مسارات التطبيقات مع User-Agent متصفح قياسي. تُنشئ خط أساس للزيارات الطبيعية في تحليلات F5 XC. |
| 02 | `02-concurrent-load.sh` | curl | طلبات HTTP متوازية باستخدام curl مع اتصالات متزامنة متعددة. تختبر التعامل مع الحمل وتُنشئ خطوط أساس للإنتاجية. |
| 03 | `03-mixed-methods.sh` | curl | أساليب HTTP مختلطة (GET, POST, PUT, DELETE) ضد نقاط نهاية API مع حمولات صالحة. تولّد أنماط زيارات API طبيعية للمقارنة مع زيارات الهجوم. |
| 04 | `04-user-journey.sh` | playwright | يحاكي رحلة مستخدم واقعية: تصفح الصفحات، والبحث عن المنتجات، وملء النماذج، والتنقل بين التطبيقات باستخدام Playwright مع إعدادات متصفح قياسية. |

**سلوك F5 XC المتوقع:** يجب أن تمر جميع الطلبات في هذه المجموعة دون تدخل WAF أو الدفاع ضد الروبوتات. استخدم هذه المجموعة لتوليد زيارات "نظيفة" في لوحة معلومات التحليلات، ثم قارن مع زيارات مجموعة الهجوم لإظهار الفرق بين أنماط الطلبات الشرعية والخبيثة.

## waf-encoding-evasion

**البرامج النصية:** 7 | **المدة المقدرة:** 5-10 دقائق

هجمات التهرب بالتشفير متعدد الطبقات المصممة لاختبار ما إذا كان WAF يقوم بتطبيع وفك تشفير الحمولات بشكل صحيح قبل الفحص. تغطي تشفير URL الفردي والمزدوج والثلاثي؛ وتشفير كيانات HTML (عشري، سداسي عشري، مسمى، محشو بالأصفار)؛ وإدراج Unicode/UTF-8 (مسافة بعرض صفري، BOM، أحرف بعرض كامل، واصلة ناعمة، تسلسلات overlong)؛ والتشفير المختلط/المتداخل متعدد الطبقات (كيانات HTML مشفرة بـ URL، URL مزدوج داخل كيانات، تراكمات ثلاثية الطبقات)؛ وحقن البايت الفارغ؛ وتشفير IIS %uXXXX؛ والتلاعب بالحالة؛ وتقسيم جسم التشفير المجزأ؛ وتهرب معامل Base64؛ وحمولات مشفرة في الرؤوس وملفات تعريف الارتباط. يستهدف أي تطبيق.

**ميزة F5 XC:** WAF (عمق تطبيع التشفير)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-url-encoding.sh` | curl | تشفير URL الفردي والمزدوج والثلاثي لحمولات SQLi وXSS واجتياز المسار عبر مسارات نقاط نهاية متعددة. يختبر ما إذا كان WAF يفك تشفير معاملات URL بشكل تكراري. |
| 02 | `02-html-entity-encoding.sh` | curl, python3 | تشفير كيانات HTML العشري والسداسي العشري والمسمى والمحشو بالأصفار لحمولات الهجوم في كل من معاملات GET وأجسام POST. يتضمن حقن القوالب عبر `&#123;&#123;7*7&#125;&#125;`. |
| 03 | `03-unicode-utf8-evasion.sh` | curl | إدراج مسافة بعرض صفري (U+200B)، وBOM (U+FEFF)، وأحرف بعرض كامل، وواصلة ناعمة، وZWNJ في كلمات SQL المفتاحية ووسوم XSS. يختبر تسلسلات UTF-8 الطويلة (2 بايت و3 بايت). |
| 04 | `04-mixed-nested-encoding.sh` | curl | حمولات متعددة الطبقات: كيانات HTML مشفرة بـ URL، وURL مزدوج داخل كيانات، وتراكمات ثلاثية الطبقات، وجسم JSON مع تهريب Unicode (`<`)، ونمط حقن القالب المحدد `{{7*7}}` للمستخدم. |
| 05 | `05-null-byte-iis-base64.sh` | curl | حقن البايت الفارغ (`%00`) لتجاوز الامتداد، وتشفير Unicode لـ IIS `%uXXXX`، وحمولات مشفرة بـ Base64 في معاملات URL مع علامات فك التشفير. |
| 06 | `06-case-chunked-evasion.sh` | curl | تعشير الحالة مع التشفير (`%3CsCrIpT%3E`)، والتشفير المجزأ لتقسيم كلمات SQL/XSS المفتاحية عبر أجزاء HTTP، وفحوصات تعارض Content-Length/Transfer-Encoding، وفواصل المسار البديلة (`%5c`، `%c0%af`). |
| 07 | `07-header-cookie-evasion.sh` | curl | حمولات هجوم مشفرة محقونة عبر رؤوس Cookie وReferer وUser-Agent وX-Forwarded-For وX-Original-URL. يختبر ما إذا كان WAF يفحص ويفك تشفير جميع حقول رؤوس HTTP، وليس فقط معاملات الاستعلام وأجسام POST. |

**سلوك F5 XC المتوقع:** يجب أن يقوم WAF المكوَّن بشكل صحيح بتطبيع جميع طبقات التشفير قبل مطابقة الأنماط. يجب حظر الحمولات ذات التشفير الفردي على الفور. تكشف اختبارات التشفير المزدوج والثلاثي عن عمق خط أنابيب فك التشفير في WAF. تكشف اختبارات إدراج Unicode ما إذا كانت الأحرف غير المرئية يتم تجريدها قبل مطابقة الكلمات المفتاحية. النتائج التي تمر فيها المتغيرات المشفرة بينما يتم حظر المكافئات النصية العادية تشير إلى فجوات التطبيع في سياسة WAF.

## web-app-attacks

**البرامج النصية:** 6 | **المدة المقدرة:** 12-20 دقيقة

اختبار ثغرات OWASP Top 10 لتطبيقات الويب ضد Juice Shop وDVWA عبر F5 XC. يتضمن حقن SQL، وXSS، وحقن الأوامر، واجتياز المسار، ومسح Nikto، والمسح القائم على قوالب Nuclei. يستهدف تطبيقي Juice Shop وDVWA.

**ميزة F5 XC:** WAF (جدار حماية تطبيقات الويب)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-sqli.sh` | sqlmap, curl | حمولات حقن SQL ضد واجهة برمجة بحث Juice Shop ونقطة نهاية SQLi في DVWA. يشغل عمليات مسح sqlmap الآلية ثم يرسل 10 حمولات حقن مباشرة عبر curl. |
| 02 | `02-xss.sh` | dalfox, curl | البرمجة النصية عبر المواقع ضد نقاط نهاية XSS المنعكس/المخزن في Juice Shop وDVWA. يشغل عمليات مسح dalfox الآلية ثم يرسل 12 حمولة XSS مباشرة تشمل وسوم script ومعالجات الأحداث واستخراج ملفات تعريف الارتباط. |
| 03 | `03-command-injection.sh` | curl | حمولات حقن أوامر نظام التشغيل ضد نقطة نهاية exec في DVWA. يرسل 12 حمولة تشمل سلاسل الأنابيب والفواصل المنقوطة والأسطر الجديدة المشفرة بـ URL وأنماط الصدفة العكسية. |
| 04 | `04-path-traversal.sh` | curl | اجتياز الدليل ضد نقاط نهاية تطبيقات متعددة. يختبر اجتياز `../` القياسي، والمتغيرات المشفرة بـ URL، والمتغيرات ذات التشفير المزدوج، ومتغيرات البايت الفارغ، ومتغيرات Unicode عبر 6 مسارات نقاط نهاية. |
| 05 | `05-nikto-scan.sh` | nikto | مسح ثغرات خادم الويب الكامل مع حد زمني 120 ثانية. يولّد طلبات HTTP متنوعة تُطلق مطابقة توقيعات WAF. |
| 06 | `06-nuclei-scan.sh` | nuclei | مسح ثغرات قائم على القوالب بمستويات خطورة متوسطة وعالية وحرجة. محدود المعدل بـ 50 طلبًا/ثانية. |

**سلوك F5 XC المتوقع:** يجب أن يحظر WAF أو يُبلِّغ عن الطلبات التي تحتوي على حمولات حقن SQL وXSS وحقن الأوامر واجتياز المسار. ستُظهر لوحة معلومات أحداث الأمان فئات الانتهاكات ومعرفات التوقيعات وتفاصيل الطلبات لكل طلب محظور.
