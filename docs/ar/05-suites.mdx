كل مجموعة حركة مرور هي دليل تحت `/opt/traffic-generator/suites/` يحتوي على برامج نصية مرقمة تُنفذ بالترتيب بواسطة `runner.sh`. يقبل كل برنامج نصي اسم FQDN المستهدف كمعامل أول ويكتب المخرجات إلى كل من stdout ودليل النتائج.

## api-attacks

**البرامج النصية:** 4 | **المدة المقدرة:** 10-17 دقيقة

اختبار OWASP API Security Top 10 ضد VAmPI عبر F5 XC. يسجل مستخدم اختبار، ويحصل على رمز مصادقة، ثم يختبر BOLA، والمصادقة بالقوة الغاشمة، والتعرض المفرط للبيانات، وتفويض مستوى الوظيفة المعطل، وأنماط SSRF، وحقن SQL ضد نقاط نهاية REST API، واكتشاف المعاملات المخفية، وتزييف نقاط النهاية. يستهدف تطبيق VAmPI.

**ميزة F5 XC:** أمان API / اكتشاف API

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-vampi-owasp-top10.sh` | curl, jq | مجموعة اختبار شاملة لـ OWASP API Top 10. يسجل مستخدم اختبار، ويحصل على رمز مصادقة، ثم يختبر BOLA (الوصول إلى بيانات مستخدمين آخرين)، والمصادقة بالقوة الغاشمة، والتعرض المفرط للبيانات، وتفويض مستوى الوظيفة المعطل، وأنماط SSRF، وفحص سوء تكوين الأمان. |
| 02 | `02-sqlmap-api.sh` | sqlmap | حقن SQL ضد نقاط نهاية VAmPI REST API: البحث عن المستخدم، وتسجيل الدخول (POST)، والتسجيل (POST) مع حمولات JSON. |
| 03 | `03-arjun-param-discovery.sh` | arjun | اكتشاف المعاملات المخفية عبر نقاط نهاية Juice Shop وDVWA وVAmPI. يكتشف معاملات الاستعلام غير الموثقة التي قد تقبل حمولات الحقن. |
| 04 | `04-ffuf-api-fuzz.sh` | ffuf | تزييف على مرحلتين: اكتشاف نقاط النهاية باستخدام قائمة كلمات لمسارات API الشائعة (api, swagger, graphql, admin, .env, .git) عبر جميع بادئات التطبيقات، ثم تزييف أساليب HTTP (GET, POST, PUT, DELETE, PATCH, OPTIONS, TRACE, PROPFIND) ضد نقاط النهاية الرئيسية. |

**السلوك المتوقع لـ F5 XC:** يجب أن يقوم اكتشاف API بتعيين نقاط نهاية VAmPI API وتحديد أنماط الطلبات غير العادية. يجب أن تضع سياسات أمان API علامة على محاولات الوصول غير المصرح به، وحمولات الحقن في أجسام JSON، وأنماط التعداد. سيعرض مخزون API في F5 XC نقاط النهاية المكتشفة مع عدد الطلبات.

## bot-simulation

**البرامج النصية:** 4 | **المدة المقدرة:** 5-10 دقائق

نشاط المتصفح الآلي وأنماط الزحف الشبيهة بالروبوتات عبر F5 XC. يشمل Chrome بدون واجهة رسومية، وPuppeteer مع إضافة التخفي، والزحف السريع، والتفاعل الآلي مع النماذج. يستهدف جميع التطبيقات.

**ميزة F5 XC:** الدفاع ضد الروبوتات

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-headless-chrome.sh` | playwright | تصفح Chromium بدون واجهة رسومية: يحمّل الهدف، وينتقل بين مسارات التطبيقات، ويستخرج عناوين الصفحات والمحتوى، ويلتقط لقطات شاشة. يولّد حركة مرور قائمة على المتصفح بدون إضافات تخفي (قابلة للكشف كأتمتة). |
| 02 | `02-puppeteer-stealth.sh` | puppeteer, puppeteer-extra-plugin-stealth | تصفح متخفٍ باستخدام puppeteer-extra مع إضافة التخفي. يحاول التهرب من كشف الروبوتات عن طريق تعديل خصائص navigator وWebGL وبصمات المتصفح الأخرى. |
| 03 | `03-rapid-crawl.sh` | curl, wget | طلبات HTTP عالية التردد باستخدام curl وwget في تتابع سريع. يحاكي سلوك الكاشط بدون تأخير بين الطلبات مع تدوير عنيف لـ User-Agent. |
| 04 | `04-form-interaction.sh` | playwright | ملء النماذج وتقديمها آلياً ضد تسجيل دخول DVWA، وتسجيل Juice Shop، ونقاط نهاية VAmPI API باستخدام واجهات برمجة التفاعل مع النماذج في Playwright. |

**السلوك المتوقع لـ F5 XC:** يجب أن يصنف الدفاع ضد الروبوتات حركة المرور كآلية بناءً على تحليل بصمة المتصفح وتوقيت الطلبات ونتائج تحدي JavaScript. يجب اكتشاف Chrome بدون واجهة رسومية بدون تخفي فوراً. يختبر Puppeteer المتخفي ما إذا كانت تقنيات التهرب المتقدمة تتجاوز الكشف. ستعرض لوحة معلومات الدفاع ضد الروبوتات فئات تصنيف الروبوتات وإجراءات التخفيف.

## cdn-load-testing

**البرامج النصية:** 18 | **المدة المقدرة:** 20-30 دقيقة

اختبار سلوك ذاكرة التخزين المؤقت لشبكة CDN وحماية الأصل. يشمل الإنتاجية الأساسية، وعزل سلسلة الاستعلام، وتنويع accept-encoding، ومحاكاة القطيع المتدافع، وتجاوز POST/PUT، وتصادم مفتاح ذاكرة التخزين المؤقت، وGET المشروط، وطلب النطاق، ومحاكاة التطهير، واستنفاد مجمع الاتصالات، وتحسين keepalive، وعبء مصافحة TLS، وتخزين الكائنات الكبيرة مؤقتاً، وتجاوز فشل الأصول المتعددة، وتحديد المعدل، واختبار نسبة gzip، وتعدد إرسال HTTP/2، ومعيار kraken المجمع. يستهدف تطبيق محاكي CDN.

**ميزة F5 XC:** تكامل CDN

## crapi-exploits

**البرامج النصية:** 16 | **المدة المقدرة:** 15-25 دقيقة

استغلال تحديات OWASP crAPI (واجهة برمجة تطبيقات سخيفة تماماً). يشمل التسجيل/المصادقة، وموقع مركبة BOLA، وتقارير ميكانيكي BOLA، والقوة الغاشمة لـ OTP، وميكانيكيات تعرض البيانات، والتعرض المفرط للبيانات، وأوامر التخصيص الجماعي، والتحقق من صحة قسيمة SSRF، وحقن NoSQL، والتلاعب بـ JWT، ولوحة معلومات IDOR، وتجاوز 2FA، والوصول إلى API الداخلي، وتلوث المعاملات من جانب الخادم، وحذف المستخدم، وIDOR لرفع الفيديو. يستهدف تطبيق crAPI.

**ميزة F5 XC:** أمان API

## csd-demo-attacks

هجمات حقن JavaScript من جانب العميل. يشمل حقن كاشط البطاقات، ومصادر النماذج، ومسجل لوحة المفاتيح، ومعدّن العملات المشفرة، واختطاف DOM. يختبر هجمات سلسلة التوريد من جانب العميل على طراز Magecart التي يكشفها ويحظرها الدفاع من جانب العميل في F5 XC. يستهدف تطبيق CSD Demo.

**ميزة F5 XC:** الدفاع من جانب العميل

## demoapp-attacks

**البرامج النصية:** 2 | **المدة المقدرة:** 2-3 دقائق

حمولات هجوم موجهة ضد نقاط نهاية اختبار WAF المدمجة في F5 DemoApp (`/WAF/SQL`، `/WAF/XSS`، `/WAF/DIR`). يرسل حمولات حقن SQL وXSS واجتياز المسار مباشرة إلى نقاط النهاية المصممة لإظهار سلوك حظر WAF. استخدم هذه المجموعة عند استهداف خادم محتوى DemoApp بدلاً من تطبيقات خادم الأصل القياسية.

**ميزة F5 XC:** WAF (جدار حماية تطبيقات الويب)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-sqli-waf-endpoint.sh` | curl, python3 | 15 حمولة حقن SQL ضد `/WAF/SQL?age=` بما في ذلك UNION SELECT وDROP TABLE وWAITFOR DELAY وxp_cmdshell وتجاوز قائم على التعليقات وحمولات المنطق البولياني. |
| 02 | `02-xss-waf-endpoint.sh` | curl, python3 | 18 حمولة XSS ضد `/WAF/XSS?update=` بما في ذلك علامات script ومعالجات الأحداث (onerror, onload, ontoggle, onfocus) وeval بترميز base64 وتعبير style وحمولات متعددة الأشكال واستخراج ملفات تعريف الارتباط. |

**السلوك المتوقع لـ F5 XC:** يجب أن يحظر WAF الطلبات التي تحتوي على حمولات حقن SQL وXSS ويعيد صفحة حظر. ستعرض لوحة معلومات أحداث الأمان فئات الانتهاكات ومطابقات التوقيعات لكل طلب محظور.

## dvga-exploits

**البرامج النصية:** 9 | **المدة المقدرة:** 8-12 دقيقة

استغلال ثغرات خاصة بـ GraphQL ضد تطبيق Damn Vulnerable GraphQL Application. يشمل هجوم حرمان الخدمة بالاستعلامات المجمعة، والتكرار العميق، وتكرار الحقول، وحقن SQL عبر الفلتر، وXSS عبر createPaste، وهجوم حرمان الخدمة القائم على الأسماء المستعارة، وإساءة استخدام الاستبطان، وتجاوز التفويض، وكشف المعلومات. يستهدف تطبيق DVGA.

**ميزة F5 XC:** أمان API (GraphQL)

## dvwa-exploits

**البرامج النصية:** 14 | **المدة المقدرة:** 15-25 دقيقة

اختبار OWASP Top 10 ضد تطبيق Damn Vulnerable Web Application. يشمل القوة الغاشمة، وحقن الأوامر، وتغيير كلمة مرور CSRF، وتضمين الملفات، ورفع الملفات، وحقن SQL (أعمى)، وحقن SQL (اتحادي)، وXSS (DOM)، وXSS (منعكس)، وXSS (مخزّن)، وتجاوز CAPTCHA، ومعرفات الجلسة الضعيفة، وCORS غير الآمن، وإعادة التوجيه المفتوح. يستهدف تطبيق DVWA.

**ميزة F5 XC:** WAF

## javascript-exploits

**البرامج النصية:** 3 | **المدة المقدرة:** 3-5 دقائق

أنماط حقن البرامج النصية من جانب العميل والتلاعب بـ DOM التي تحفز كشف الدفاع من جانب العميل. يشمل حقن البرامج النصية المضمنة التي تحاكي كاشطات على طراز Magecart، والتلاعب بـ DOM عبر Chromium بدون واجهة رسومية، ومحاكاة البرامج النصية لجهات خارجية. يستهدف تطبيق CSD Demo.

**ميزة F5 XC:** الدفاع من جانب العميل (CSD)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-inline-script-injection.sh` | curl | يرسل طلبات تحتوي على حمولات JavaScript مضمنة مصممة لمحاكاة كاشطات بطاقات الائتمان على طراز Magecart وبرامج استخراج البيانات. |
| 02 | `02-dom-manipulation.sh` | playwright | يستخدم Chromium بدون واجهة رسومية لتحميل صفحة CSD التوضيحية وحقن تعديلات DOM: إضافة علامات script، وتعديل إجراءات النماذج، وإدراج إطارات iframe غير مرئية تستخرج بيانات النماذج. |
| 03 | `03-third-party-script-sim.sh` | curl, playwright | يحاكي البرامج النصية المخترقة لجهات خارجية عن طريق حقن طلبات تشير إلى مصادر JavaScript خارجية ومحاولة تحميل برامج نصية غير مصرح بها في سياق الصفحة. |

**السلوك المتوقع لـ F5 XC:** يجب أن يكشف الدفاع من جانب العميل عن تعديلات البرامج النصية غير المصرح بها على الصفحة المحمية. ستعرض لوحة معلومات CSD تنبيهات لمصادر البرامج النصية الجديدة وعناصر DOM المعدلة ومحاولات استخراج البيانات. يتحقق هذا من وظيفة المرحلة الثانية من CSD.

<Aside type="note">
مجموعة javascript-exploits مصممة للعمل جنباً إلى جنب مع تطبيق CSD التوضيحي على خادم الأصل. تأكد من تمكين المسار `/csd-demo/` على خادم الأصل قبل تشغيل هذه المجموعة.
</Aside>

## juice-shop-exploits

**البرامج النصية:** 12 | **المدة المقدرة:** 10-18 دقيقة

استغلال تحديات OWASP Juice Shop. يشمل تجاوز تسجيل الدخول بحقن SQL، واتحاد بحث حقن SQL، وXSS DOM المنعكس، وXSS المخزن عبر API، ووصول سلة IDOR، ووصول قسم المسؤول، والتعليقات المزورة، والوصول إلى الملفات بالبايت الفارغ، وXSS المنعكس في البحث، وحقن رؤوس HTTP، والتسجيل المتكرر، والقوة الغاشمة لتسجيل الدخول. يستهدف تطبيق Juice Shop.

**ميزة F5 XC:** WAF، الدفاع ضد الروبوتات

## mitre-attack

**البرامج النصية:** 8 | **المدة المقدرة:** 10-15 دقيقة

محاكاة تكتيكات إطار عمل MITRE ATT&CK. يشمل الاستطلاع (المسح الخارجي)، والوصول الأولي (حشو بيانات الاعتماد)، والتنفيذ (حقن الأوامر)، والوصول إلى بيانات الاعتماد (كلمات المرور الافتراضية)، والاكتشاف (تعداد الأدلة)، ومحاكاة الحركة الجانبية، والجمع (كشط البيانات)، ومحاكاة الاستخراج. يعيّن كل برنامج نصي لتكتيك MITRE ATT&CK محدد لتقارير تغطية التهديدات المنظمة. يستهدف جميع التطبيقات.

**ميزة F5 XC:** WAF، الدفاع ضد الروبوتات، أمان API

**ميزة F5 XC:** WAF، مسح تطبيقات الويب

**السلوك المتوقع لـ F5 XC:** يولّد مسح SSL العديد من مصافحات TLS مع مقترحات مجموعات تشفير غير عادية. بينما لا يحظر F5 XC هذه عادةً، يكون نمط حركة المرور مرئياً في سجلات الوصول. القيمة الأساسية هي التحقق من أن تكوين TLS لـ F5 XC يلبي أفضل ممارسات الأمان.

## traffic-generation

**البرامج النصية:** 4 | **المدة المقدرة:** 5-10 دقائق (قابلة للتكوين)

حركة مرور HTTP مشروعة عالية الحجم لقياس خط الأساس واختبار التحميل. يؤسس أنماط حركة مرور عادية في تحليلات F5 XC للمقارنة مع حركة مرور مجموعة الهجمات. يستهدف جميع التطبيقات.

**ميزة F5 XC:** الكل (مقارنة خط الأساس)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-baseline-http.sh` | curl | طلبات HTTP GET متتالية لجميع مسارات التطبيقات مع User-Agent متصفح قياسي. يؤسس خط أساس لحركة المرور العادية في تحليلات F5 XC. |
| 02 | `02-concurrent-load.sh` | curl | طلبات HTTP متوازية باستخدام curl مع اتصالات متزامنة متعددة. يختبر التعامل مع التحميل ويؤسس خطوط أساس الإنتاجية. |
| 03 | `03-mixed-methods.sh` | curl | أساليب HTTP مختلطة (GET, POST, PUT, DELETE) ضد نقاط نهاية API مع حمولات صالحة. يولّد أنماط حركة مرور API عادية للمقارنة مع حركة مرور الهجمات. |
| 04 | `04-user-journey.sh` | playwright | يحاكي رحلة مستخدم واقعية: تصفح الصفحات، والبحث عن المنتجات، وملء النماذج، والتنقل بين التطبيقات باستخدام Playwright مع إعدادات المتصفح القياسية. |

**السلوك المتوقع لـ F5 XC:** يجب أن تمر جميع الطلبات في هذه المجموعة دون تدخل WAF أو الدفاع ضد الروبوتات. استخدم هذه المجموعة لتوليد حركة مرور "نظيفة" في لوحة معلومات التحليلات، ثم قارنها مع حركة مرور مجموعة الهجمات لإظهار الفرق بين أنماط الطلبات المشروعة والخبيثة.

## waf-encoding-evasion

**البرامج النصية:** 7 | **المدة المقدرة:** 5-10 دقائق

هجمات التهرب بالترميز متعدد الطبقات مصممة لاختبار ما إذا كان WAF يقوم بتطبيع الحمولات وفك ترميزها بشكل صحيح قبل الفحص. تغطي ترميز URL مفرد ومزدوج وثلاثي؛ وترميز كيانات HTML (عشري، سداسي عشري، مسمى، مبطن بالأصفار)؛ وإدراج Unicode/UTF-8 (مسافة بعرض صفري، BOM، أحرف بعرض كامل، واصلة ناعمة، تسلسلات مطولة)؛ والترميز المختلط/المتداخل متعدد الطبقات (كيانات HTML مرمزة بـ URL، URL مزدوج داخل الكيانات، مكدسات ثلاثية الطبقات)؛ وحقن البايت الفارغ؛ وترميز IIS %uXXXX؛ والتلاعب بحالة الأحرف؛ وتقسيم جسم ترميز النقل المجزأ؛ وتهرب معاملات Base64؛ وحمولات مرمزة في الرؤوس/ملفات تعريف الارتباط. يستهدف أي تطبيق.

**ميزة F5 XC:** WAF (عمق تطبيع الترميز)

| الترتيب | البرنامج النصي | الأدوات | الوصف |
|---|---|---|---|
| 01 | `01-url-encoding.sh` | curl | ترميز URL مفرد ومزدوج وثلاثي لحمولات SQLi وXSS واجتياز المسار عبر مسارات نقاط نهاية متعددة. يختبر ما إذا كان WAF يفك ترميز معاملات URL بشكل تكراري. |
| 02 | `02-html-entity-encoding.sh` | curl, python3 | ترميز كيانات HTML عشري وسداسي عشري ومسمى ومبطن بالأصفار لحمولات الهجوم في كل من معاملات GET وأجسام POST. يشمل حقن القوالب عبر `&#123;&#123;7*7&#125;&#125;`. |
| 03 | `03-unicode-utf8-evasion.sh` | curl | إدراج مسافة بعرض صفري (U+200B)، وBOM (U+FEFF)، وأحرف بعرض كامل، وواصلة ناعمة، وZWNJ في كلمات SQL المفتاحية وعلامات XSS. يختبر تسلسلات UTF-8 المطولة (2 بايت و3 بايت). |
| 04 | `04-mixed-nested-encoding.sh` | curl | حمولات متعددة الطبقات: كيانات HTML مرمزة بـ URL، وURL مزدوج داخل الكيانات، ومكدسات ثلاثية الطبقات، وجسم JSON مع هروب Unicode (`<`)، ونمط حقن القالب `{{7*7}}` الخاص بالمستخدم. |
| 05 | `05-null-byte-iis-base64.sh` | curl | حقن البايت الفارغ (`%00`) لتجاوز الامتداد، وترميز Unicode لـ IIS `%uXXXX`، وحمولات مرمزة بـ Base64 في معاملات URL مع علامات فك الترميز. |
| 06 | `06-case-chunked-evasion.sh` | curl | عشوائية حالة الأحرف مدمجة مع الترميز (`%3CsCrIpT%3E`)، وترميز النقل المجزأ لتقسيم كلمات SQL/XSS المفتاحية عبر أجزاء HTTP، وتحقيقات تعارض Content-Length/Transfer-Encoding، وفواصل مسار بديلة (`%5c`، `%c0%af`). |
| 07 | `07-header-cookie-evasion.sh` | curl | حمولات هجوم مرمزة محقونة عبر رؤوس Cookie وReferer وUser-Agent وX-Forwarded-For وX-Original-URL. يختبر ما إذا كان WAF يفحص ويفك ترميز جميع حقول رؤوس HTTP، وليس فقط معاملات الاستعلام وأجسام POST. |

**السلوك المتوقع لـ F5 XC:** يجب أن يقوم WAF المكوّن بشكل صحيح بتطبيع جميع طبقات الترميز قبل مطابقة الأنماط. يجب حظر الحمولات المرمزة مرة واحدة فوراً. تكشف اختبارات الترميز المزدوج والثلاثي عن عمق خط أنابيب فك الترميز في WAF. تكشف اختبارات إدراج Unicode ما إذا كانت الأحرف غير المرئية تُزال قبل مطابقة الكلمات المفتاحية. النتائج التي تمر فيها المتغيرات المرمزة بينما يتم حظر المكافئات بالنص العادي تشير إلى فجوات في التطبيع في سياسة WAF.

## web-app-attacks
