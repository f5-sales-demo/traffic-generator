## สิ่งที่ระบบนี้มอบให้

Traffic Generator คือ Azure VM ที่สร้างขึ้นเฉพาะสำหรับผลิตทราฟฟิกการโจมตีที่สมจริง การสแกนตรวจสอบ การจำลองบอท และรูปแบบการใช้งาน API ในทางที่ผิดต่อ F5 Distributed Cloud HTTP load balancer ระบบนี้ตรวจสอบว่านโยบาย WAF, Bot Defense, API Security และ Client-Side Defense ได้รับการกำหนดค่าอย่างถูกต้องโดยการสร้างทราฟฟิกที่ตรงกับสิ่งที่ฟีเจอร์เหล่านั้นได้รับการออกแบบมาเพื่อตรวจจับและบล็อก

เครื่องมือทั้งหมดถูกติดตั้งล่วงหน้าผ่าน cloud-init ระหว่างการจัดเตรียมด้วย Terraform ทราฟฟิกถูกจัดระเบียบเป็นชุดที่สามารถรันทีละชุดหรือรันตามลำดับโดยใช้ตัวจัดการ `runner.sh` ที่รวมมาให้

## หมวดหมู่ชุดทราฟฟิก

| ชุด | คำอธิบาย | ฟีเจอร์ F5 XC ที่ได้รับการตรวจสอบ |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10, โหมด SQLMap API, การค้นหาพารามิเตอร์, การ fuzz endpoint | API Security |
| [bot-simulation](/traffic-generator/05-suites/#bot-simulation) | Headless Chrome, Puppeteer stealth, ระบบอัตโนมัติ Playwright, การ crawl แบบรวดเร็ว | Bot Defense |
| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | พฤติกรรมแคช, thundering herd, connection pool, HTTP/2 multiplexing | CDN Integration |
| [crapi-exploits](/traffic-generator/05-suites/#crapi-exploits) | BOLA, OTP bruteforce, การจัดการ JWT, SSRF, NoSQL injection, IDOR | API Security |
| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | Card skimmer, formjacker, keylogger, cryptominer, DOM hijack | Client-Side Defense |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | Batch query DoS, deep recursion, SQL injection, introspection abuse | API Security (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | Brute force, command injection, CSRF, file inclusion, SQLi, XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | การจัดการ DOM, การฉีดสคริปต์แบบ inline, เพย์โหลดสกิมมิงสไตล์ Magecart | Client-Side Defense |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | SQLi login bypass, XSS, IDOR, การเข้าถึงแอดมิน, การเข้าถึงไฟล์ด้วย null byte | WAF, Bot Defense |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | กลยุทธ์ ATT&CK: การสำรวจ, การเข้าถึงเบื้องต้น, การเข้าถึงข้อมูลรับรอง, การขโมยข้อมูล | WAF, Bot Defense, API Security |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | การสแกนช่องโหว่ด้วย ZAP, Nikto, Nuclei, Nmap, รายงาน OWASP รวม | WAF, Web App Scanning |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | การเพิ่ม concurrency, โหลดต่อเนื่อง, การทดสอบ spike, การค้นหาจุดแตกหัก | DDoS, Rate Limiting |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap, Masscan, Gobuster, Subfinder, การ brute-force ไดเรกทอรี | WAF / Bot Defense |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA, BOPLA, BFLA, การหลีกเลี่ยง rate limiting, JWT weak secret | API Security |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | การวิเคราะห์การกำหนดค่า TLS ด้วย SSLScan, sslyze, testssl.sh | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | ทราฟฟิก HTTP ที่ถูกต้องปริมาณสูงสำหรับ baseline และการทดสอบโหลด | All |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | การเข้ารหัส URL/HTML/Unicode หลายชั้น, การเข้ารหัสซ้อนแบบผสม, chunked TE, header injection | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | SQL injection, XSS, command injection, path traversal, Nikto, Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi, XSS, path traversal ต่อ endpoint ทดสอบ WAF ของ F5 DemoApp | WAF |
