เครื่องมือทุกรายการด้านล่างนี้ได้รับการติดตั้งล่วงหน้าระหว่างการจัดเตรียมผ่าน cloud-init ไม่จำเป็นต้องติดตั้งด้วยตนเองหลังจากการ deploy

## การทดสอบเว็บแอปพลิเคชัน

| เครื่องมือ | วิธีการติดตั้ง | คำสั่งตัวอย่าง | ใช้โดยชุด |
|---|---|---|---|
| nikto | apt | `nikto -h https://TARGET -maxtime 120s` | web-app-attacks |
| sqlmap | apt | `sqlmap --batch -u "https://TARGET/dvwa/vulnerabilities/sqli/?id=1"` | web-app-attacks, api-attacks |
| nuclei | ghlatest | `nuclei -u https://TARGET -severity medium,high,critical` | web-app-attacks |
| dalfox | ghlatest | `dalfox url "https://TARGET/search?q=test" --silence` | web-app-attacks |
| ffuf | ghlatest | `ffuf -u https://TARGET/FUZZ -w wordlist.txt -mc all -fc 404` | api-attacks |
| gobuster | ghlatest | `gobuster dir -u https://TARGET -w /opt/SecLists/Discovery/Web-Content/common.txt` | reconnaissance |
| feroxbuster | ghlatest | `feroxbuster -u https://TARGET -w /opt/SecLists/Discovery/Web-Content/raft-medium-directories.txt` | reconnaissance |
| dirb | apt | `dirb https://TARGET /usr/share/dirb/wordlists/common.txt` | reconnaissance |
| whatweb | apt | `whatweb https://TARGET` | reconnaissance |
| wfuzz | pip | `wfuzz -c -z file,wordlist.txt -e urlencode,double_urlencode "https://TARGET/FUZZ"` | waf-encoding-evasion |

## การทดสอบการหลบเลี่ยง WAF

| เครื่องมือ | วิธีการติดตั้ง | คำสั่งตัวอย่าง | ใช้โดยชุด |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

GoTestWAF (โดย Wallarm) เป็นเฟรมเวิร์กประเมิน WAF ตามมาตรฐานอุตสาหกรรมที่ทวีคูณ payload ข้ามตัวเข้ารหัสหลายรูปแบบ (URL, Base64, JSUnicode) และโซนการวาง (body, header, URL param, cookie) waf-bypass (โดย Nemesida) ส่ง payload ด้วยการเข้ารหัส Base64, HTML-ENTITY และ UTF-16 ข้ามโซน ARGS, BODY, COOKIE และ HEADER

## เฉพาะระดับ Full เท่านั้น

เครื่องมือเหล่านี้จะติดตั้งเมื่อ `tool_tier = "full"` เท่านั้น:

| เครื่องมือ | วิธีการติดตั้ง | คำสั่งตัวอย่าง | ใช้โดยชุด |
|---|---|---|---|
| ZAP (OWASP Zed Attack Proxy) | ghlatest | `zap-cli quick-scan -s all -r https://TARGET` | web-app-attacks |
| Metasploit Framework | installer script | `msfconsole -q -x "use auxiliary/scanner/http/http_version; set RHOSTS TARGET; run; exit"` | reconnaissance |

:::note
ZAP และ Metasploit เพิ่มพื้นที่ดิสก์ประมาณ 1.5 GiB และเวลาการจัดเตรียมเพิ่มขึ้น 5-10 นาที ใช้ระดับ full เฉพาะเมื่อคุณต้องการเครื่องมือเหล่านี้สำหรับการสาธิตเท่านั้น
:::

## ข้อมูลอ้างอิงวิธีการติดตั้ง

| วิธีการ | คำอธิบาย |
|---|---|
| apt | ติดตั้งจากที่เก็บ APT ของ Ubuntu 24.04 ผ่าน `apt-get install` |
| ghlatest | ดาวน์โหลดไบนารีล่าสุดจาก GitHub Releases โดยใช้ `ghlatest.sh` และ `install-release.sh` |
| uv | ติดตั้งเป็นเครื่องมือ Python โดยใช้ตัวจัดการแพ็กเกจ `uv` |
| pip | ติดตั้งเป็นแพ็กเกจ Python โดยใช้ `pip install` ลงใน Python ของระบบ |
| npm | ติดตั้งเป็นแพ็กเกจ Node.js แบบ global โดยใช้ `npm install -g` |
| go install | คอมไพล์จากซอร์สโค้ดโดยใช้ `go install` และติดตั้งไปที่ `/usr/local/bin` |
| git clone | โคลนที่เก็บไปยังไดเรกทอรี `/opt/` |
| git clone + pip | โคลนที่เก็บและติดตั้ง dependency ของ Python |
| installer script | สคริปต์ตัวติดตั้งจากผู้จำหน่ายที่ถูกเรียกใช้ระหว่าง cloud-init |
