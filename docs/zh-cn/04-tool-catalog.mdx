## WAF 绕过测试

| 工具 | 安装方式 | 示例命令 | 所属套件 |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

GoTestWAF（由 Wallarm 开发）是业界标准的 WAF 评估框架，能够通过多种编码器（URL、Base64、JSUnicode）和放置区域（body、header、URL 参数、cookie）对载荷进行组合扩展。waf-bypass（由 Nemesida 开发）使用 Base64、HTML-ENTITY 和 UTF-16 编码，跨 ARGS、BODY、COOKIE 和 HEADER 区域发送载荷。

## 安装方式参考

| 方式 | 说明 |
|---|---|
| apt | 通过 `apt-get install` 从 Ubuntu 24.04 APT 仓库安装 |
| ghlatest | 使用 `ghlatest.sh` 和 `install-release.sh` 从 GitHub Releases 下载最新二进制文件 |
| uv | 使用 `uv` 包管理器作为 Python 工具安装 |
| pip | 使用 `pip install` 安装到系统 Python 中的 Python 包 |
| npm | 使用 `npm install -g` 安装的全局 Node.js 包 |
| go install | 使用 `go install` 从源码编译并安装到 `/usr/local/bin` |
| git clone | 仓库克隆到 `/opt/` 目录 |
| git clone + pip | 克隆仓库并安装 Python 依赖 |
| installer script | 在 cloud-init 期间执行供应商提供的安装脚本 |
