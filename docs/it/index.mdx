## Cosa Offre

Il Traffic Generator è una VM Azure appositamente progettata che produce traffico di attacco realistico, scansioni di ricognizione, simulazione di bot e pattern di abuso API contro un HTTP load balancer F5 Distributed Cloud. Valida che le policy WAF, Bot Defense, API Security e Client-Side Defense siano correttamente configurate generando esattamente il traffico che tali funzionalità sono progettate per rilevare e bloccare.

Tutti gli strumenti sono preinstallati tramite cloud-init durante il provisioning Terraform. Il traffico è organizzato in suite che possono essere eseguite singolarmente o in sequenza utilizzando l'orchestratore `runner.sh` incluso.
