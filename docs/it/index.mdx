## Cosa Offre

Il Traffic Generator è una VM Azure appositamente progettata che produce traffico di attacco realistico, scansioni di ricognizione, simulazione di bot e pattern di abuso delle API contro un HTTP load balancer F5 Distributed Cloud. Valida che le policy WAF, Bot Defense, API Security e Client-Side Defense siano correttamente configurate generando esattamente il traffico che queste funzionalità sono progettate per rilevare e bloccare.

Tutti gli strumenti sono pre-installati tramite cloud-init durante il provisioning Terraform. Il traffico è organizzato in suite che possono essere eseguite individualmente o in sequenza utilizzando l'orchestratore `runner.sh` incluso.

## Categorie delle Suite di Traffico

| Suite | Descrizione | Funzionalità F5 XC Validata |
|---|---|---|
| [api-attacks](/traffic-generator/05-suites/#api-attacks) | OWASP API Top 10, SQLMap modalità API, scoperta parametri, fuzzing degli endpoint | API Security |

| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | Comportamento cache, thundering herd, pool di connessioni, multiplexing HTTP/2 | Integrazione CDN |

| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | Card skimmer, formjacker, keylogger, cryptominer, DOM hijack | Client-Side Defense |

| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | Forza bruta, command injection, CSRF, file inclusion, SQLi, XSS | WAF |

| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | Scansione vulnerabilità con ZAP, Nikto, Nuclei, Nmap, report OWASP combinato | WAF, Scansione Web App |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | Rampa di concorrenza, carico sostenuto, test di picco, scoperta del punto di rottura | DDoS, Rate Limiting |

| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | Encoding multi-livello URL/HTML/Unicode, encoding annidato misto, chunked TE, header injection | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | SQL injection, XSS, command injection, path traversal, Nikto, Nuclei | WAF |
