## 本ツールの概要

Traffic Generatorは、F5 Distributed Cloud HTTPロードバランサーに対して、リアルな攻撃トラフィック、偵察スキャン、ボットシミュレーション、API悪用パターンを生成する専用のAzure VMです。WAFポリシー、Bot Defense、APIセキュリティ、Client-Side Defenseが正しく構成されていることを、それらの機能が検出・ブロックするように設計されたトラフィックを正確に生成することで検証します。

すべてのツールは、Terraformプロビジョニング時にcloud-initを介してプリインストールされます。トラフィックはスイートごとに整理されており、付属の`runner.sh`オーケストレーターを使用して個別または順次実行できます。
