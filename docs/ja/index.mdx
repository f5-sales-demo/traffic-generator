## 本ツールが提供するもの

トラフィックジェネレーターは、F5 Distributed Cloud HTTPロードバランサーに対してリアルな攻撃トラフィック、偵察スキャン、ボットシミュレーション、およびAPI悪用パターンを生成するために構築された専用のAzure VMです。WAFポリシー、Bot Defense、APIセキュリティ、およびClient-Side Defenseが正しく設定されていることを、それらの機能が検出・ブロックするように設計されたトラフィックを正確に生成することで検証します。

すべてのツールは、Terraformプロビジョニング時にcloud-initを介して事前インストールされます。トラフィックはスイートごとに整理されており、付属の `runner.sh` オーケストレーターを使用して個別または順番に実行できます。
