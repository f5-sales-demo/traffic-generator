## O Que Isto Oferece

O Gerador de Tráfego é uma VM Azure construída especificamente para produzir tráfego de ataque realista, varreduras de reconhecimento, simulação de bots e padrões de abuso de API contra um balanceador de carga HTTP do F5 Distributed Cloud. Ele valida que as políticas de WAF, Bot Defense, API Security e Client-Side Defense estão corretamente configuradas, gerando exatamente o tráfego que essas funcionalidades foram projetadas para detectar e bloquear.

Todas as ferramentas são pré-instaladas via cloud-init durante o provisionamento do Terraform. O tráfego é organizado em suítes que podem ser executadas individualmente ou em sequência usando o orquestrador `runner.sh` incluído.
