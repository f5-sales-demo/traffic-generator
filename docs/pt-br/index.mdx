## O Que Isto Fornece

O Gerador de Tráfego é uma VM Azure construída especificamente para produzir tráfego de ataque realista, varreduras de reconhecimento, simulação de bots e padrões de abuso de API contra um balanceador de carga HTTP do F5 Distributed Cloud. Ele valida que as políticas de WAF, Bot Defense, API Security e Client-Side Defense estão corretamente configuradas ao gerar exatamente o tráfego que essas funcionalidades são projetadas para detectar e bloquear.

Todas as ferramentas são pré-instaladas via cloud-init durante o provisionamento com Terraform. O tráfego é organizado em suítes que podem ser executadas individualmente ou em sequência usando o orquestrador `runner.sh` incluído.

| Suíte | Descrição | Funcionalidade F5 XC Validada |
|---|---|---|

| [cdn-load-testing](/traffic-generator/05-suites/#cdn-load-testing) | Comportamento de cache, thundering herd, pool de conexões, multiplexação HTTP/2 | Integração CDN |

| [csd-demo-attacks](/traffic-generator/05-suites/#csd-demo-attacks) | Card skimmer, formjacker, keylogger, cryptominer, sequestro de DOM | Client-Side Defense |
| [dvga-exploits](/traffic-generator/05-suites/#dvga-exploits) | DoS por batch query, recursão profunda, injeção SQL, abuso de introspection | API Security (GraphQL) |
| [dvwa-exploits](/traffic-generator/05-suites/#dvwa-exploits) | Força bruta, injeção de comandos, CSRF, inclusão de arquivos, SQLi, XSS | WAF |
| [javascript-exploits](/traffic-generator/05-suites/#javascript-exploits) | Manipulação de DOM, injeção de script inline, payloads de skimming estilo Magecart | Client-Side Defense |
| [juice-shop-exploits](/traffic-generator/05-suites/#juice-shop-exploits) | Bypass de login SQLi, XSS, IDOR, acesso admin, acesso a arquivos com null byte | WAF, Bot Defense |
| [mitre-attack](/traffic-generator/05-suites/#mitre-attack) | Táticas ATT&CK: reconhecimento, acesso inicial, acesso a credenciais, exfiltração | WAF, Bot Defense, API Security |
| [owasp-scanning](/traffic-generator/05-suites/#owasp-scanning) | ZAP, Nikto, Nuclei, varredura de vulnerabilidades Nmap, relatório OWASP combinado | WAF, Varredura de Aplicações Web |
| [performance-testing](/traffic-generator/05-suites/#performance-testing) | Rampa de concorrência, carga sustentada, teste de pico, descoberta de ponto de ruptura | DDoS, Rate Limiting |
| [reconnaissance](/traffic-generator/05-suites/#reconnaissance) | Nmap, Masscan, Gobuster, Subfinder, força bruta de diretórios | WAF / Bot Defense |
| [restaurant-exploits](/traffic-generator/05-suites/#restaurant-exploits) | BOLA, BOPLA, BFLA, bypass de rate limiting, segredo fraco JWT | API Security |
| [ssl-scanning](/traffic-generator/05-suites/#ssl-scanning) | SSLScan, sslyze, testssl.sh análise de configuração TLS | WAF |
| [traffic-generation](/traffic-generator/05-suites/#traffic-generation) | Tráfego HTTP legítimo de alto volume para baseline e teste de carga | Todos |
| [waf-encoding-evasion](/traffic-generator/05-suites/#waf-encoding-evasion) | Codificação multicamada URL/HTML/Unicode, codificação aninhada mista, chunked TE, injeção de cabeçalhos | WAF |
| [web-app-attacks](/traffic-generator/05-suites/#web-app-attacks) | Injeção SQL, XSS, injeção de comandos, path traversal, Nikto, Nuclei | WAF |
| [demoapp-attacks](/traffic-generator/05-suites/#demoapp-attacks) | SQLi, XSS, path traversal contra endpoints de teste WAF do F5 DemoApp | WAF |
