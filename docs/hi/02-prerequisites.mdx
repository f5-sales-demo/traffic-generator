## आवश्यक

### Azure सदस्यता

निम्नलिखित बनाने के लिए Contributor भूमिका या समकक्ष अनुमतियों के साथ एक सक्रिय Azure सदस्यता:

- रिसोर्स ग्रुप
- वर्चुअल नेटवर्क और सबनेट
- नेटवर्क सुरक्षा समूह
- पब्लिक IP पते
- वर्चुअल मशीनें (डिफ़ॉल्ट Standard_F16s_v2)

### Azure CLI

अपनी सदस्यता ID प्राप्त करने के लिए Azure CLI इंस्टॉल करें और प्रमाणित करें:

```bash
curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash

az login

az account show --query '{name:name, id:id}' -o table
```

`id` मान कॉपी करें -- यह Terraform द्वारा आवश्यक `subscription_id` है।

### Terraform

Terraform >= 1.5 आवश्यक है:

```bash
terraform version
```

यदि इंस्टॉल नहीं है, तो HashiCorp APT रिपॉज़िटरी का उपयोग करें:

```bash
wget -O- https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list
sudo apt update && sudo apt install terraform
```

### SSH कुंजी युग्म

VM एक्सेस के लिए एक SSH कुंजी युग्म। Ed25519 अनुशंसित है:

```bash
ssh-keygen -t ed25519 -f ~/.ssh/traffic-generator-key -N ""
```

### नेटवर्क एक्सेस

ट्रैफ़िक जेनरेटर VM को प्रोविज़निंग के दौरान निम्नलिखित स्रोतों से टूल डाउनलोड करने के लिए आउटबाउंड HTTPS एक्सेस की आवश्यकता है:

- GitHub Releases (nuclei, dalfox, ffuf, gobuster, feroxbuster, subfinder, httpx, amass)
- PyPI (sqlmap, scapy, impacket, arjun, pwntools, mitmproxy, sslyze, playwright)
- npm रजिस्ट्री (puppeteer, puppeteer-extra-plugin-stealth)
- Ubuntu APT रिपॉज़िटरीज़ (nikto, nmap, masscan, hydra, tshark, और अन्य सिस्टम पैकेज)
- Git रिपॉज़िटरीज़ (testssl.sh, recon-ng, spiderfoot, SecLists)

यदि आपकी Azure सदस्यता आउटबाउंड फ़ायरवॉल नियम या NSG प्रतिबंध लागू करती है, तो सुनिश्चित करें कि प्रारंभिक प्रोविज़निंग अवधि (15-25 मिनट) के दौरान इन स्रोतों तक HTTPS (पोर्ट 443) की अनुमति है।

### पूर्व-तैनात घटक

ट्रैफ़िक जेनरेटर तैनात करने से पहले, आपके पास होना चाहिए:

1. **ओरिजिन सर्वर** तैनात और पहुँच योग्य हो (देखें [origin-server डॉक्स](https://f5xc-salesdemos.github.io/origin-server/03-deploy/))
2. **F5 XC HTTP लोड बैलेंसर** ओरिजिन सर्वर को बैकएंड के रूप में कॉन्फ़िगर किया गया हो
3. लोड बैलेंसर से जुड़ी एक **WAF पॉलिसी** (WAF ट्रैफ़िक सत्यापन के लिए)

`target_fqdn` Terraform वेरिएबल F5 XC लोड बैलेंसर के पब्लिक डोमेन पर सेट होना चाहिए।

:::tip
जब लैब उपयोग में न हो तो चल रहे शुल्कों से बचने के लिए `terraform destroy` का उपयोग करें। प्रक्रिया के लिए [विघटन](../08-teardown/) देखें।
:::

:::note
Standard_F16s_v2 (32 GiB RAM) डिफ़ॉल्ट है क्योंकि सुइट निष्पादन के दौरान कई टूल एक साथ चलते हैं। हेडलेस Chromium के साथ ब्राउज़र ऑटोमेशन (Playwright, Puppeteer) प्राथमिक मेमोरी उपभोक्ता है। छोटे F-सीरीज़ SKU गैर-ब्राउज़र सुइट्स के लिए काम करेंगे लेकिन बॉट-सिमुलेशन चलाते समय तंग हो सकते हैं।
:::
