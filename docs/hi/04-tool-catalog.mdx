## WAF इवेज़न टेस्टिंग

| टूल | इंस्टॉल विधि | उदाहरण कमांड | सूट द्वारा उपयोग |
|---|---|---|---|
| gotestwaf | go install | `cd /opt/gotestwaf && gotestwaf --url https://TARGET --noEmailReport --includePayloads=false` | waf-encoding-evasion |
| waf-bypass | git clone + pip | `waf-bypass --host TARGET:443 --timeout 10 --threads 10` | waf-encoding-evasion |

GoTestWAF (Wallarm द्वारा) एक उद्योग-मानक WAF मूल्यांकन फ्रेमवर्क है जो कई एनकोडर्स (URL, Base64, JSUnicode) और प्लेसमेंट ज़ोन (body, header, URL param, cookie) में पेलोड्स को गुणा करता है। waf-bypass (Nemesida द्वारा) Base64, HTML-ENTITY, और UTF-16 एन्कोडिंग के साथ ARGS, BODY, COOKIE, और HEADER ज़ोन में पेलोड्स भेजता है।

## केवल फुल टियर

ये टूल्स केवल तभी इंस्टॉल किए जाते हैं जब `tool_tier = "full"` हो:

| टूल | इंस्टॉल विधि | उदाहरण कमांड | सूट द्वारा उपयोग |
|---|---|---|---|
| ZAP (OWASP Zed Attack Proxy) | ghlatest | `zap-cli quick-scan -s all -r https://TARGET` | web-app-attacks |
| Metasploit Framework | installer script | `msfconsole -q -x "use auxiliary/scanner/http/http_version; set RHOSTS TARGET; run; exit"` | reconnaissance |

:::note
ZAP और Metasploit डिस्क फुटप्रिंट में लगभग 1.5 GiB और प्रोविजनिंग समय में 5-10 मिनट जोड़ते हैं। फुल टियर का उपयोग केवल तभी करें जब आपको डेमो के लिए विशेष रूप से इन टूल्स की आवश्यकता हो।
:::

## इंस्टॉल विधि संदर्भ

| विधि | विवरण |
|---|---|
| apt | `apt-get install` के माध्यम से Ubuntu 24.04 APT रिपॉज़िटरी से इंस्टॉल किया गया |
| ghlatest | `ghlatest.sh` और `install-release.sh` का उपयोग करके GitHub Releases से नवीनतम बाइनरी डाउनलोड की गई |
| uv | `uv` पैकेज मैनेजर का उपयोग करके Python टूल के रूप में इंस्टॉल किया गया |
| pip | `pip install` का उपयोग करके सिस्टम Python में Python पैकेज के रूप में इंस्टॉल किया गया |
| npm | `npm install -g` का उपयोग करके ग्लोबल Node.js पैकेज के रूप में इंस्टॉल किया गया |
| go install | `go install` का उपयोग करके सोर्स से कंपाइल किया गया और `/usr/local/bin` में इंस्टॉल किया गया |
| git clone | `/opt/` डायरेक्टरी में रिपॉज़िटरी क्लोन की गई |
| git clone + pip | रिपॉज़िटरी क्लोन की गई और Python डिपेंडेंसीज़ इंस्टॉल की गईं |
| installer script | cloud-init के दौरान वेंडर द्वारा प्रदत्त इंस्टॉलर स्क्रिप्ट निष्पादित की गई |
