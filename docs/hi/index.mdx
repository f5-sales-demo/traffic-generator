## यह क्या प्रदान करता है

ट्रैफ़िक जनरेटर एक विशेष रूप से निर्मित Azure VM है जो F5 Distributed Cloud HTTP लोड बैलेंसर के विरुद्ध यथार्थवादी अटैक ट्रैफ़िक, रिकॉनेसेंस स्कैन, बॉट सिमुलेशन, और API दुरुपयोग पैटर्न उत्पन्न करता है। यह सत्यापित करता है कि WAF पॉलिसीज़, Bot Defense, API Security, और Client-Side Defense सही ढंग से कॉन्फ़िगर हैं, उन सुविधाओं द्वारा पहचाने और अवरुद्ध किए जाने के लिए डिज़ाइन किए गए सटीक ट्रैफ़िक को उत्पन्न करके।

सभी उपकरण Terraform प्रोविज़निंग के दौरान cloud-init के माध्यम से पूर्व-स्थापित होते हैं। ट्रैफ़िक सूट्स में व्यवस्थित किया जाता है जिन्हें शामिल `runner.sh` ऑर्केस्ट्रेटर का उपयोग करके व्यक्तिगत रूप से या क्रम में चलाया जा सकता है।
