## 제공 기능

트래픽 생성기는 F5 Distributed Cloud HTTP 로드 밸런서를 대상으로 실제적인 공격 트래픽, 정찰 스캔, 봇 시뮬레이션, API 남용 패턴을 생성하기 위해 특별히 제작된 Azure VM입니다. WAF 정책, Bot Defense, API Security, Client-Side Defense가 올바르게 구성되었는지 검증하기 위해 해당 기능들이 탐지하고 차단하도록 설계된 정확한 트래픽을 생성합니다.

모든 도구는 Terraform 프로비저닝 중에 cloud-init을 통해 사전 설치됩니다. 트래픽은 개별적으로 또는 포함된 `runner.sh` 오케스트레이터를 사용하여 순차적으로 실행할 수 있는 스위트로 구성되어 있습니다.
